# 디자인 설정 가이드 (커스터마이징)

이 가이드는 코딩 없이 병원의 아이덴티티에 맞춰 홈페이지의 디자인을 변경하고 관리하는 방법을 설명합니다.

## 1. 브랜딩 컬러 및 폰트
- **메인 컬러**: 병원의 로고 또는 상징색에 맞춰 홈페이지의 주요 포인트 컬러(버튼, 강조 텍스트 등)를 변경할 수 있습니다.
- **타이포그래피**: 가독성이 좋은 추천 폰트 세트 중에서 선택하여 병원의 톤앤매너를 설정합니다.

## 2. 로고 및 이미지 관리
- **로고 업로드**: 상단 네비게이션과 하단 푸터에 표시될 로고 이미지를 업로드합니다.
- **파비콘(Favicon)**: 브라우저 탭에 표시될 작은 아이콘을 설정합니다.

## 3. 메인 배너 및 섹션 순서
- **드래그 앤 드랍**: 홈페이지 메인에 노출될 섹션(비주얼 배너, 진료 프로그램, 리뷰 등)의 순서를 클릭만으로 변경할 수 있습니다.
- **섹션 활성화**: 필요한 섹션만 골라서 노출하거나 일시적으로 숨길 수 있습니다.

## 4. 모바일 최적화
- 디자인 변경 사항은 모바일 화면에서도 실시간으로 최적화되어 적용됩니다. '미리보기' 기능을 통해 기기별 화면을 확인해보세요.

> [!NOTE]
> 디자인 변경 후 '저장하기'를 누르면 실제 홈페이지에 즉시 반영됩니다. 큰 변경 전에는 반드시 '미리보기'로 먼저 확인하는 것이 좋습니다.
