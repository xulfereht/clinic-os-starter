## 3. 에이전트와 협업하는 방식

**헤드라인:** 말하면 됩니다. 에이전트가 합니다.

**4가지 협업 예시:**

| 상황 | 원장님이 하는 일 | 에이전트가 하는 일 |
|------|----------------|------------------|
| **경영 분석** | "이번 달 매출 분석해줘" | 환자별·프로그램별 분석 + 추천 액션 |
| **콘텐츠 제작** | "허리통증 블로그 글 써줘" | 원장님 톤으로 초안 작성 + 광고심의 검토 |
| **환자 관리** | "60일 이상 안 온 환자 리마인드" | 대상 추출 + 메시지 초안 + 발송 |
| **홈편이지 개선** | "메인 문구 바꿔줘" | 변경 + 빌드 + 배포 완료 |

## 5. 왜 AIMO인가

**헤드라인:** 기술이 아닌, 파트너십

**차별점 3가지:**

**원장님이 주체입니다**
- 병원 경영의 주도권은 항상 원장님에게
- AI는 판단을 돕고 실행을 돕는 파트너
- 기술이 아닌 "경영자의 도메인 날리지"가 레버리지

**데이터가 쌓일수록 똑똑해집니다**
- 환자 패턴 분석, 이탈 예측, 시간대별 최적화
- 쓸수록 정확해지는 조언
- 새 기능도 자동으로 업데이트

**완전한 소유권**
- 월 구독료 없음, 플랫폼 종속 없음
- 내 Cloudflare 계정에서 직접 운영
- 소스코드 수정도 자유롭게

## 7. FAQ

**Q. 코딩을 몰라도 되나요?**
A. 네. 코드를 볼 일이 없습니다. 에이전트에게 대화로 지시하면 됩니다.

**Q. 어떤 병원에 적합한가요?**
A. 한의원을 중심으로 시작했지만, 현재는 내과, 피부과, 성형외과 등 다양한 병원에서 활용 중입니다.

**Q. 비용은 얼마인가요?**
A. Clinic-OS는 Cloudflare 무료 플랜으로 운영됩니다 (별도 월 구독료 없음). AI 에이전트(Claude Pro) 구독만 별도입니다(월 $20, 선택). 워크숍은 프로그램별 상이합니다.

**Q. 기존 홈페이지가 있는데요?**
A. 네이버 블로그, 플레이스 정보를 자동으로 가져옵니다. 기존 콘텐츠를 잃지 않습니다.

**Q. 업데이트는 어떻게 되나요?**
A. 새 기능(스킬)과 시스템 개선이 자동으로 배포됩니다. 내 커스터마이징은 그대로 보존됩니다.
