# 백록담한의원 블로그 콘텐츠 작성 가이드

## 기본 원칙

### 가독성
- 문단 길이: 200자 이내 (모바일 화면 기준 3-4줄)
- 총 길이: 2,000~4,000자 (읽기 시간 8-12분)
- H2 섹션: 4-7개 (너무 세분화하지 말 것)

### 구조
1. **훅 오프닝** (1-2문장) - 문제 제기
2. **사례 소개** - 가상의 환자 스토리
3. **원인 분석** - 3가지 주요 원인
4. **전문적 설명** - 한의학 + 현대의학 관점
5. **해결책** - 단계별 접근
6. **결론/CTA** - 행동 유도

## 마크다운 스타일 규칙

### 제목
- H1: 사용 금지 (페이지 제목으로 자동 생성)
- H2: ## (메인 섹션) - 4-7개
- H3: ### (서브 섹션) - 필요시만

### 강조
- **볼드**: 중요 개념, 키워드 (남용 금지)
- <u>밑줄</u>: 강조가 필요한 문구
- _이탤릭_: 한의학 용어 (처음 등장시)

### 표
- 3열 구조 권장 (특징|증상|해결책)
- 모바일에서 가로스크롤됨을 고려
- 간결한 내용 (셀당 1-2줄)

### 인용/사례 블록
```markdown
| [내용] |
| ------ |
```

### 이미지
```markdown
![설명 텍스트](https://barun-static.s3.ap-northeast-2.amazonaws.com/blog/snowhitebambi/...)
```
- 이미지 아래 설명은 자동으로 캡션 처리됨

## 금지 사항

❌ **과장된 표현**
- "완벽하게", "100%", "즉시", "영구적으로"
- "최고의", "유일한", " revolutionary"

❌ **의료 광고법 위반**
- 특정 질환의 완치 보장
- 경쟁사와의 직접 비교
- 환자의 구체적 실명/사진 (동의 없이)

❌ **가독성 해치는 것**
- 500자 이상의 장문 문단
- 10개 이상의 H2 섹션
- 과도한 볼드/색상 사용
