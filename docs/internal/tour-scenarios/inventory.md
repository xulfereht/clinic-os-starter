# 재고 관리 (/admin/inventory) — 투어 시나리오

## 페이지 목적
원내 소모품 재고 현황을 한눈에 파악하고 입고/출고를 기록하는 통합 관리 시스템입니다.

## 시나리오 (7 스텝)

### Step 1: 재고 현황 진입 및 부족 알림 확인
- **title**: 재고 관리 시작하기
- **text**: 이 페이지는 병원에서 사용하는 모든 소모품의 재고를 관리하는 중심지입니다. 최소 재고 이하인 품목은 페이지 맨 위의 빨간 경고 배너에 표시됩니다. 경고 배너의 '확인하기' 링크를 클릭하면 해당 품목으로 자동 스크롤됩니다. 재고 현황 탭에서는 전체 품목 목록을 보고, 거래처 관리 탭에서는 공급처 정보를 관리할 수 있습니다.
- **highlight**: `.mb-6.bg-red-50.border.border-red-200.rounded-xl` (경고 배너)
- **trigger**: click
- **tips**:
  - 최소 재고 기준은 각 품목마다 다르게 설정할 수 있습니다
  - 부족 품목이 많으면 5개만 표시되고 "+N건" 형식으로 나머지가 표시됩니다
  - 부족 알림은 자동으로 업데이트됩니다
- **buttons**:
  - selector: `button[id="openItemBtn"]`
  - label: 품목 추가
  - desc: 새로운 소모품 품목을 등록합니다

### Step 2: 품목 검색 및 카테고리 필터
- **title**: 특정 품목 빠르게 찾기
- **text**: 테이블 위의 검색창에 품목명을 입력하면 실시간으로 해당 품목을 검색할 수 있습니다. 또한 카테고리 드롭다운을 사용해 특정 카테고리(의약품, 소모품, 기구 등)의 품목만 필터링할 수 있습니다. 검색과 필터를 조합해 원하는 품목을 빠르게 찾을 수 있으며, 현재 보이는 품목 개수가 우측 아래에 표시됩니다.
- **highlight**: `#itemSearch` (검색창)
- **trigger**: input
- **tips**:
  - 검색은 품목명으로만 작동합니다
  - 카테고리 필터와 검색을 함께 사용할 수 있습니다
  - 필터링 결과는 자동으로 갱신됩니다
- **buttons**:
  - selector: `#categoryFilter`
  - label: 카테고리 필터
  - desc: 특정 카테고리의 품목만 표시합니다

### Step 3: 재고 수량 확인 및 상태 파악
- **title**: 각 품목의 재고 상태 확인
- **text**: 테이블의 가장 오른쪽 열은 현재 재고 수량을 컬러 배지로 표시합니다. 초록색 배지는 충분한 재고를 의미하고, 빨간색 배지는 최소 재고 이하라는 경고입니다. 배지에 표시된 숫자는 현재 재고 개수이며, 옆의 단위(ea, 개, 박스 등)는 그 품목의 거래 단위입니다.
- **highlight**: `.text-center` (재고 수량 열)
- **trigger**: hover
- **tips**:
  - 초록색 = 충분한 재고 (최소 기준 이상)
  - 빨간색 = 부족한 재고 (최소 기준 이하)
  - 재고 수량은 입고/출고 때마다 자동 업데이트됩니다
- **buttons**: 없음 (상태 표시만)

### Step 4: 품목 입고 기록
- **title**: 새 소모품 입고하기
- **text**: 테이블의 각 행에 마우스를 올리면 오른쪽에 '+ 입고' 버튼이 나타납니다. 이를 클릭하면 입고 모달이 열려 새로 들어온 물량을 기록할 수 있습니다. 입고 수량, 입고 날짜, 거래처 정보, 비용 등을 입력합니다. 입고 기록이 추가되면 해당 품목의 재고 수량이 자동으로 증가합니다.
- **highlight**: `.restock-btn` (+ 입고 버튼)
- **trigger**: click
- **tips**:
  - 입고 수량은 현재 재고에 더해집니다
  - 입고 날짜를 정확히 기록하면 나중에 입고 이력을 추적할 수 있습니다
  - 거래처와 비용 정보는 나중의 발주 관리에 도움이 됩니다
- **buttons**:
  - selector: `.restock-btn`
  - label: + 입고
  - desc: 품목의 입고 수량을 기록합니다

### Step 5: 품목 출고 기록
- **title**: 사용한 소모품 출고하기
- **text**: 각 행의 '- 사용' 버튼을 클릭하면 출고 모달이 열립니다. 여기서 사용한 소모품의 수량을 입력합니다. 출고 기록이 추가되면 해당 품목의 재고 수량이 자동으로 감소합니다. 의약품이나 일회용품처럼 진료 중에 사용되는 품목의 경우 정확한 출고 기록이 비용 관리에 중요합니다.
- **highlight**: `.usage-btn` (- 사용 버튼)
- **trigger**: click
- **tips**:
  - 출고 수량은 현재 재고에서 차감됩니다
  - 재고가 0 이하로 내려가지 않도록 주의하세요
  - 정기적인 출고 기록으로 소비 패턴을 분석할 수 있습니다
  - 음수(-) 수량은 입력할 수 없습니다
- **buttons**:
  - selector: `.usage-btn`
  - label: - 사용
  - desc: 사용한 품목의 출고 수량을 기록합니다

### Step 6: 거래처 관리
- **title**: 공급처 정보 관리하기
- **text**: 페이지 상단의 탭에서 '거래처 관리'를 선택하면 병원의 모든 공급처(거래처) 정보를 볼 수 있습니다. 거래처명, 담당자, 연락처, 카테고리(의약품 도매, 의료기구, 소모품 등)를 관리합니다. '거래처 추가' 버튼으로 새로운 공급처를 등록할 수 있고, 각 거래처별로 발주 이력을 추적할 수 있습니다.
- **highlight**: `button[class*="tab-btn"]` (탭 버튼)
- **trigger**: click
- **tips**:
  - 거래처는 품목 등록 시 선택할 수 있습니다
  - 주요 거래처는 저장해두면 품목 추가 시 선택 드롭다운에 나타납니다
  - 거래처 연락처는 빠른 발주 시 유용합니다
- **buttons**:
  - selector: `#openVendorBtn`
  - label: 거래처 추가
  - desc: 새로운 공급처를 등록합니다

### Step 7: 품목 추가
- **title**: 새로운 소모품 품목 등록하기
- **text**: 우측 상단의 파란 '품목 추가' 버튼을 클릭하면 새 품목 등록 모달이 열립니다. 품목명, 카테고리, 거래처, 현재 재고 수량, 최소 재고 기준, 보관 위치 등을 입력합니다. 이렇게 등록한 품목은 목록에 추가되어 매일 관리할 수 있습니다.
- **highlight**: `#openItemBtn` (품목 추가 버튼)
- **trigger**: click
- **tips**:
  - 품목명은 정확하게 입력하면 나중에 검색할 때 편합니다
  - 카테고리를 정확히 지정하면 필터링에 유리합니다
  - 최소 재고 기준은 평상시 사용량과 발주 기간을 고려해 설정하세요
  - 보관 위치는 입고/출고 시 물품을 찾기 쉽게 하기 위함입니다
- **buttons**:
  - selector: `button[type="submit"]`
  - label: 등록
  - desc: 새로운 품목을 재고 목록에 추가합니다
