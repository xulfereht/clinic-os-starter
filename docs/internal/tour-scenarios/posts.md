# 콘텐츠 관리 (/admin/posts) — 투어 시나리오

## 페이지 목적
블로그, 칼럼, 후기 등의 콘텐츠를 검색, 필터링, 조회하고 편집 또는 발행합니다. 데스크톱에서는 테이블 보기, 모바일에서는 카드 보기로 자동 전환됩니다.

---

## 시나리오 (15 스텝)

### Step 1: 페이지 입장 및 용어 이해
- **title**: 콘텐츠 관리 페이지 소개
- **text**: 이 페이지에서는 한의원의 블로그 글, 칼럼, 환자 후기(리뷰)를 관리합니다. 각 글은 제목, 유형, 카테고리, 작성자, 발행 상태, 작성일을 보여주며, 클릭하여 편집할 수 있습니다.
- **highlight**: `.max-w-5xl` 컨테이너
- **trigger**: confirm
- **tips**:
  - 블로그 = 의료진이 작성한 건강 정보, 치료법 가이드
  - 칼럼 = 의료진의 개인 의견, 칼럼성 글
  - 후기 = 환자가 작성한 후기 또는 추천글

### Step 2: 상단 헤더 및 매뉴얼
- **title**: 페이지 헤더 및 가이드 링크
- **text**: 페이지 제목 "콘텐츠 관리"와 설명, 외부 매뉴얼 링크(HQ 가이드)가 표시됩니다. 매뉴얼을 클릭하면 전체 기능 설명을 볼 수 있습니다.
- **highlight**: `PageHeader` 컴포넌트 영역
- **trigger**: confirm
- **tips**:
  - 매뉴얼은 별도 탭에서 열립니다
  - 콘텐츠 작성, SEO 최적화, 발행 등에 대한 상세 가이드를 참고하세요

### Step 3: 새 글 작성 버튼
- **title**: 새로운 콘텐츠 작성
- **text**: 우상단의 파란 버튼 "+ 글쓰기"를 클릭하면 글 작성 페이지로 이동합니다. 여기서 제목, 내용, 카테고리, 발행 상태를 설정하여 새 글을 작성합니다.
- **highlight**: `a[href="/admin/posts/new"]`
- **trigger**: click
- **buttons**:
  - selector: `a[href="/admin/posts/new"]`
  - label: "+ 글쓰기"
  - desc: "새로운 블로그 글이나 칼럼을 작성합니다"
- **tips**:
  - 매번 새 글을 작성할 때 이 버튼을 사용합니다
  - 작성 후 "임시저장" 또는 "발행"을 선택할 수 있습니다

### Step 4: 필터 영역 소개
- **title**: 필터 및 검색 기능
- **text**: 필터 섹션에는 4개 필터가 있습니다: 검색(제목), 유형(블로그/칼럼), 카테고리(다이어트/통증/피부 등), 상태(발행됨/임시저장). 각 필터를 조합하여 원하는 글을 찾을 수 있습니다.
- **highlight**: `#filterForm`
- **trigger**: confirm
- **tips**:
  - 필터는 독립적으로 적용 가능합니다
  - 드롭다운 선택 시 자동으로 폼이 제출되어 결과를 필터링합니다
  - 검색어는 제목에서만 검색됩니다 (내용 검색 불가)

### Step 5: 검색 입력 필드
- **title**: 제목으로 글 검색
- **text**: "검색" 입력 필드에 제목의 일부를 입력합니다 (예: "비만"). 0.5초 후 자동으로 검색이 실행되고 일치하는 글들이 표시됩니다.
- **highlight**: `input[name="q"]`
- **trigger**: input
- **buttons**:
  - selector: `input[name="q"]`
  - label: "검색"
  - desc: "글 제목으로 검색합니다 (자동 필터링)"
- **tips**:
  - 공백을 포함하여 검색 가능합니다
  - 대문자/소문자 구분 없음
  - 입력 중 자동으로 검색되므로 엔터를 누를 필요 없습니다
  - 검색 결과는 페이지 상단의 전체 개수와 현재 범위가 표시됩니다

### Step 6: 유형 필터 (블로그/칼럼)
- **title**: 글의 유형별로 필터링
- **text**: "유형" 드롭다운에서 "전체", "블로그", "칼럼"을 선택합니다. 선택하면 즉시 해당 유형의 글만 표시됩니다.
- **highlight**: `select[name="type"]`
- **trigger**: click
- **buttons**:
  - selector: `select[name="type"]`
  - label: "유형"
  - desc: "블로그, 칼럼, 후기 등의 유형으로 필터링합니다"
- **tips**:
  - "전체"를 선택하면 모든 유형을 보여줍니다
  - 드롭다운 변경 시 자동으로 폼이 제출됩니다
  - 현재 선택된 값은 회색 배경으로 강조됩니다

### Step 7: 카테고리 필터
- **title**: 진료 과목(카테고리)별로 필터링
- **text**: "카테고리" 드롭다운에서 진료 분야를 선택합니다: 다이어트, 통증, 피부질환, 여성질환, 소화기, 신경정신, 소아청소년, 보약/웰니스, 두면부. 선택하면 해당 카테고리의 글만 표시됩니다.
- **highlight**: `select[name="category"]`
- **trigger**: click
- **buttons**:
  - selector: `select[name="category"]`
  - label: "카테고리"
  - desc: "진료 과목별로 글을 필터링합니다"
- **tips**:
  - "전체"는 카테고리가 지정되지 않은 글도 포함합니다
  - 한 번에 하나의 카테고리만 선택 가능합니다
  - 글 작성 시 반드시 카테고리를 지정해야 여기서 필터될 수 있습니다

### Step 8: 상태 필터 (발행됨/임시저장)
- **title**: 발행 상태별로 필터링
- **text**: "상태" 드롭다운에서 "전체", "발행됨", "임시저장"을 선택합니다. "발행됨"은 웹사이트에 공개된 글, "임시저장"은 작성 중이거나 비공개 상태의 글입니다.
- **highlight**: `select[name="status"]`
- **trigger**: click
- **buttons**:
  - selector: `select[name="status"]`
  - label: "상태"
  - desc: "발행 상태별로 필터링합니다"
- **tips**:
  - "발행됨" 글만 홈페이지의 블로그 목록에 표시됩니다
  - "임시저장" 글은 관리자만 볼 수 있으며, 언제든 발행 또는 삭제 가능합니다
  - 상태 변경은 글 편집 페이지에서 합니다

### Step 9: 데스크톱 보기 — 테이블 헤더
- **title**: 데스크톱 테이블 구조 (큰 화면)
- **text**: 데스크톱(lg 이상)에서는 테이블 형식으로 글이 표시됩니다. 7개 열: 제목, 유형, 카테고리, 작성자, 상태, 작성일, 액션. 각 열을 통해 글의 모든 정보를 한눈에 확인할 수 있습니다.
- **highlight**: `table thead`
- **trigger**: confirm
- **tips**:
  - 모바일(sm 이하)에서는 테이블이 숨겨지고 카드 보기로 전환됩니다
  - 각 행은 호버 시 배경이 밝아져 상호작용이 가능함을 시각적으로 표시합니다

### Step 10: 제목 클릭하여 편집 페이지 이동
- **title**: 글 제목을 클릭하여 편집
- **text**: 테이블의 "제목" 열(또는 모바일 카드의 제목)을 클릭하면 글 편집 페이지(/admin/posts/{id})로 이동합니다. 여기서 제목, 내용, 카테고리, 발행 상태 등을 수정합니다.
- **highlight**: `a[href*="/admin/posts/"]`
- **trigger**: click
- **buttons**:
  - selector: `table tbody tr td:nth-child(1) a`
  - label: "제목"
  - desc: "글 편집 페이지로 이동합니다"
- **tips**:
  - 제목은 파란색으로 표시되고, 호버 시 더 진한 파란색으로 변합니다
  - 제목이 길면 2줄까지 표시되고 말줄임표로 생략됩니다
  - "수정" 액션 버튼을 클릭해도 같은 페이지로 이동합니다

### Step 11: 테이블 데이터 열 해석
- **title**: 테이블의 각 열 의미
- **text**: 
  - **제목**: 글의 제목 (클릭 시 편집)
  - **유형**: 색깔 배지로 표시 (블로그=파란색, 칼럼=보라색, 후기=주황색)
  - **카테고리**: 진료 과목명 (예: "다이어트", "-"=미지정)
  - **작성자**: 의료진 또는 환자명 (이니셜 아바타 + 이름)
  - **상태**: 발행됨(초록색)/임시저장(회색) 배지
  - **작성일**: YYYY.MM.DD 형식
  - **액션**: "수정" 버튼
- **highlight**: `table tbody tr`
- **trigger**: confirm
- **tips**:
  - 작성자가 지정되지 않으면 "미지정"으로 표시됩니다
  - 카테고리가 없으면 "-"로 표시됩니다
  - 모든 시간 정보는 서울 타임존(Asia/Seoul)을 기준으로 표시됩니다

### Step 12: 모바일 카드 보기
- **title**: 모바일/태블릿 보기 (카드 형식)
- **text**: 모바일(sm 이하)에서는 테이블이 숨겨지고 각 글이 카드 형식으로 표시됩니다. 카드에는 상태 배지, 유형 배지, 카테고리, 제목, 작성자, 작성일, 수정 버튼이 표시됩니다.
- **highlight**: `.lg\:hidden #postsMobile`
- **trigger**: confirm
- **tips**:
  - 카드는 세로 방향으로 쌓입니다
  - 각 카드는 호버 시 그림자가 커져 상호작용 가능함을 표시합니다
  - 모바일에서도 제목이나 "수정" 버튼을 클릭하면 편집 페이지로 이동합니다

### Step 13: 페이지네이션
- **title**: 여러 페이지의 글 조회
- **text**: 글이 많으면 페이지네이션이 표시됩니다. 하단의 "← 이전", "현재페이지 / 전체페이지", "다음 →" 버튼을 사용하여 다른 페이지의 글을 봅니다. 페이지당 15개 글이 표시됩니다.
- **highlight**: `.mt-4 .flex`
- **trigger**: click
- **buttons**:
  - selector: `a[href*="page="]`
  - label: "← 이전 / 다음 →"
  - desc: "다른 페이지의 글을 조회합니다"
- **tips**:
  - 페이지네이션은 모든 필터가 유지된 상태에서 작동합니다 (검색어, 카테고리, 상태 등)
  - 현재 페이지는 파란 배경으로 강조됩니다
  - 첫 페이지에서는 "← 이전" 버튼이 비활성화되고, 마지막 페이지에서는 "다음 →" 버튼이 비활성화됩니다

### Step 14: 총 게시물 개수 표시
- **title**: 전체 글의 개수 및 현재 범위
- **text**: 페이지네이션 위에 "총 N개 중 X - Y"가 표시됩니다. 예: "총 47개 중 1 - 15" = 전체 47개 글 중 현재 1~15번째 글을 보고 있다는 의미입니다.
- **highlight**: `.text-xs.text-slate-500`
- **trigger**: confirm
- **tips**:
  - 이 정보는 필터 적용 후의 결과 개수입니다
  - 따라서 검색이나 필터를 적용하면 개수가 변합니다
  - 예: 카테고리 "다이어트" 필터 시 "총 12개 중..."로 표시될 수 있습니다

### Step 15: 글이 없을 때의 표시
- **title**: 빈 목록 상태
- **text**: 글이 하나도 없거나 필터 결과가 없을 때는 중앙에 큰 아이콘(📝)과 함께 "게시물이 없습니다. 새로운 글을 작성해보세요." 메시지가 표시되고, "+ 글쓰기" 버튼이 있습니다.
- **highlight**: `tr[colspan="7"]`, `.bg-white.rounded-2xl.shadow-sm.border.border-slate-200.p-8`
- **trigger**: confirm
- **tips**:
  - 이 상태는 정상입니다. 새로운 콘텐츠는 언제든 추가할 수 있습니다
  - "+ 글쓰기" 버튼을 클릭하여 첫 글을 작성하세요
  - 필터를 조정하면 숨겨진 글들이 다시 나타날 수 있습니다

---

## 통합 워크플로우 예시

1. **새 글 작성**: "+ 글쓰기" → 제목/내용/카테고리 입력 → 임시저장 또는 발행
2. **기존 글 수정**: 제목 클릭 또는 "수정" 버튼 → 편집 페이지 → 변경 후 저장
3. **글 찾기**: 검색어 입력 → 또는 카테고리/상태 필터 적용 → 결과 표시
4. **여러 페이지 조회**: 필터 적용 → 페이지네이션으로 다른 페이지 이동
5. **발행 상태 변경**: 글 편집 → "발행" 또는 "임시저장" 선택 → 저장

## 자주 묻는 상황

| 상황 | 해결 방법 |
|------|---------|
| 작성한 글이 홈페이지에 안 보여요 | 발행 상태가 "발행됨"인지 확인하세요. "임시저장"이면 공개되지 않습니다 |
| 검색이 느려요 | 검색어가 있을 때마다 0.5초 후 자동으로 검색됩니다. 잠시 기다려주세요 |
| 제목이 길면 어떻게 되나요 | 테이블에서는 2줄까지 표시되고 말줄임표로 생략됩니다. 전체 제목은 편집 페이지에서 확인 가능합니다 |
| 카테고리를 지정하지 않은 글이 있어요 | 필터에서 "전체"를 선택하면 볼 수 있습니다. 지정하지 않은 글들은 카테고리 열에 "-"로 표시됩니다 |
| 페이지가 여러 개일 때 모든 글을 보고 싶어요 | 페이지네이션 버튼으로 모든 페이지를 이동하며 조회하거나, 필터를 해제하면 첫 페이지의 최근 글들이 표시됩니다 |
