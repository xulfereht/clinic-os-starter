# 업무 체크리스트 (/admin/tasks) — 투어 시나리오

## 페이지 목적
병원 직원의 업무를 칸반 보드 방식으로 관리하고 진행 상황을 한눈에 추적합니다.

## 시나리오 (7 스텝)

### Step 1: 칸반 보드 진입
- **title**: 업무 관리 칸반 보드 이해하기
- **text**: 이 페이지는 병원의 모든 업무를 3개의 칸(칼럼)으로 나누어 관리합니다. 왼쪽부터 '할 일(To Do)', '진행 중(In Progress)', '완료(Done)' 칸입니다. 각 칸의 헤더에는 그 상태의 업무 개수가 표시됩니다. 카드는 드래그해서 다른 칸으로 이동할 수 있어 업무 진행 상황을 시각적으로 관리할 수 있습니다.
- **highlight**: `.flex-1.min-w-[350px].flex.flex-col.bg-slate-50` (칼럼)
- **trigger**: click
- **tips**:
  - 3개 칸은 각각 색상으로 구분됩니다(회색, 파란색, 초록색)
  - 각 칸의 헤더 숫자는 실시간으로 업데이트됩니다
  - 스크롤로 많은 업무를 관리할 수 있습니다
- **buttons**:
  - selector: `button[onclick="openCreateModal()"]`
  - label: 새 업무 추가
  - desc: 새로운 업무를 등록합니다

### Step 2: 업무 카드 구조 이해하기
- **title**: 각 업무 카드의 정보 확인
- **text**: 각 업무 카드에는 업무 제목, 반복 주기(매일, 매주, 매월 등), 담당자 정보, 그리고 서브태스크 진행률 바가 표시됩니다. 빨간색 배지는 1회성 업무, 보라색은 매일 반복, 남색은 매주 반복, 분홍색은 매월 반복 업무를 의미합니다. 완료된 업무는 흐린 텍스트로 표시되어 시각적으로 구분됩니다.
- **highlight**: `.bg-white.p-4.rounded-xl.shadow-sm` (업무 카드)
- **trigger**: hover
- **tips**:
  - 반복 주기 배지는 자동으로 다음 실행 날짜를 계산합니다
  - 서브태스크는 보다 세부적인 작업 단계를 나타냅니다
  - 진행률 바는 백분율로 진행 정도를 표시합니다
- **buttons**: 없음 (정보 표시만)

### Step 3: 업무 카드 드래그해서 상태 변경
- **title**: 업무를 다른 칸으로 드래그하기
- **text**: 업무 카드를 마우스로 클릭한 후 드래그해서 다른 칼럼으로 이동할 수 있습니다. 예를 들어 '할 일'의 카드를 '진행 중' 칸으로 드래그하면 그 업무의 상태가 즉시 변경됩니다. 드래그 중에는 카드의 외곽선이 강조되어 어느 칸으로 이동할지 명확하게 표시됩니다.
- **highlight**: `div[draggable="true"]`
- **trigger**: click
- **tips**:
  - 드래그 앤 드롭은 마우스로만 지원됩니다(터치 X)
  - 드래그 중 칸 위에서 떨어지면 원래 위치로 돌아갑니다
  - 상태 변경은 즉시 서버에 저장됩니다
  - 다른 사용자가 같은 업무를 수정하면 실시간 반영됩니다
- **buttons**: 없음 (드래그 앤 드롭만)

### Step 4: 업무 상세 보기 및 서브태스크 관리
- **title**: 업무 카드를 클릭해서 상세 정보 확인
- **text**: 업무 카드를 클릭하면 상세 모달이 열립니다. 여기서 전체 내용, 담당자, 기한, 서브태스크 목록 등을 볼 수 있습니다. 서브태스크는 체크박스 형태로 되어 있어서, 각 작업 단계를 하나씩 체크하면 진행률 바가 자동으로 업데이트됩니다. 이렇게 세부 진행 상황을 추적할 수 있습니다.
- **highlight**: `.group-hover:text-blue-600` (카드)
- **trigger**: click
- **tips**:
  - 클릭 시 드래그되지 않도록 주의하세요
  - 서브태스크는 독립적으로 체크/언체크할 수 있습니다
  - 모든 서브태스크를 완료하면 진행률이 100%가 됩니다
  - 모달에서 업무 제목과 설명도 편집할 수 있습니다
- **buttons**:
  - selector: `button[onclick="closeViewModal()"]`
  - label: 닫기
  - desc: 상세 모달을 닫습니다

### Step 5: 담당자 지정 및 변경
- **title**: 업무 담당자 지정하기
- **text**: 업무 카드의 하단에는 담당자 정보가 작은 원형 배지로 표시됩니다. 담당자의 이름 첫 글자가 배지 안에 표시되고, 옆에 이름이 함께 보입니다. 상세 모달에서 담당자를 변경할 수 있고, 미지정된 업무는 "미지정" 텍스트가 회색으로 표시됩니다. 담당자를 지정하면 그 직원에게 업무가 자동으로 할당됩니다.
- **highlight**: `.flex.items-center.gap-1.5.bg-slate-100` (담당자 배지)
- **trigger**: hover
- **tips**:
  - 같은 업무를 여러 명에게 할당할 수는 없습니다
  - 담당자가 없는 업무는 "미지정"으로 표시됩니다
  - 담당자 변경도 즉시 반영됩니다
- **buttons**:
  - selector: `select[name="assignee"]`
  - label: 담당자 선택
  - desc: 업무를 수행할 담당자를 지정합니다

### Step 6: 반복 업무 관리
- **title**: 매일/매주/매월 반복되는 업무
- **text**: 업무 카드의 반복 주기 배지가 색상으로 표시됩니다. '1회성'(회색)은 한 번만, '매일'(보라색)은 매일 반복, '매주'(남색)은 매주, '매월'(분홍색)은 매월 같은 날에 자동으로 반복됩니다. 반복 업무가 완료되면 다음 실행일을 기준으로 자동으로 새 카드가 생성됩니다.
- **highlight**: `.px-2.py-0.5.rounded-md.font-medium` (반복 배지)
- **trigger**: hover
- **tips**:
  - 반복 업무는 시스템이 자동으로 관리합니다
  - 반복 주기는 상세 모달에서 변경할 수 있습니다
  - 1회성으로 변경하면 반복이 중단됩니다
  - 지난 날의 반복 업무도 볼 수 있습니다
- **buttons**:
  - selector: `select[name="frequency"]`
  - label: 반복 주기 선택
  - desc: 업무의 반복 주기를 설정합니다

### Step 7: 새 업무 등록
- **title**: 새로운 업무 추가하기
- **text**: 우측 상단의 파란 '+ 새 업무 추가' 버튼을 클릭하면 새 업무 생성 모달이 열립니다. 업무 제목(필수), 설명, 담당자, 기한, 반복 주기, 서브태스크 등을 입력합니다. 최소한 제목만 입력해도 '할 일' 칸에 새 업무가 추가되며, 나머지 정보는 나중에 상세 모달에서 수정할 수 있습니다.
- **highlight**: `button[onclick="openCreateModal()"]`
- **trigger**: click
- **tips**:
  - 업무 제목은 명확하고 구체적으로 입력하세요
  - 기한을 설정하면 만료 알림을 받을 수 있습니다
  - 반복 주기는 나중에 변경할 수 있습니다
  - 서브태스크는 쉼표로 구분해 여러 개를 입력할 수 있습니다
  - 새 업무는 항상 '할 일' 칸에 추가됩니다
- **buttons**:
  - selector: `button[type="submit"]`
  - label: 추가
  - desc: 새 업무를 칸반 보드에 추가합니다
