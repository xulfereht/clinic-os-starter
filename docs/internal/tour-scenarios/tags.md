# Tags (/admin/settings/tags) — 투어 시나리오

## 페이지 목적
환자를 분류하기 위한 태그를 카테고리별로 관리하고, 색상 지정으로 시각적 구분을 제공합니다.

## 시나리오 (6 스텝)

### Step 1: 페이지 진입 및 현황 확인
- **title**: 태그 관리 페이지 개요
- **text**: 이 페이지에서는 모든 환자 태그를 보고 관리합니다. 상단에 현재 등록된 태그 개수가 표시되고, 카테고리별로 그룹화되어 나타납니다.
- **highlight**: `.max-w-5xl`
- **trigger**: none
- **tips**: 
  - 태그는 카테고리로 자동 구분됨
  - 각 카테고리는 카드로 표시됨
  - 색상으로 태그를 빠르게 식별
- **buttons**: []

### Step 2: 검색 기능 사용
- **title**: 태그 검색
- **text**: 검색창에 태그명을 입력하면 실시간으로 일치하는 태그를 필터링합니다. 대소문자 구분 없이 부분 검색도 지원합니다.
- **highlight**: `#tagSearch`
- **trigger**: input
- **tips**:
  - 입력할 때마다 즉시 필터링됨
  - 카테고리도 함께 필터됨 (검색 결과 없으면 카테고리 숨김)
  - 검색 초기화는 입력 필드 비우기
- **buttons**: 
  - `#tagSearch / 검색 입력 / 태그명을 입력하여 필터링`

### Step 3: 새 태그 추가 버튼
- **title**: 새 태그 추가
- **text**: 우측 상단의 "태그 추가" 파란색 버튼을 클릭하면 모달 대화창이 열립니다. 여기서 새로운 태그를 추가할 수 있습니다.
- **highlight**: `#addTagBtn`
- **trigger**: click
- **tips**:
  - 버튼 옆에 현재 태그 개수 표시
  - 태그가 없으면 "첫 번째 태그 추가하기" 버튼도 표시됨
- **buttons**:
  - `#addTagBtn / 태그 추가 / 새로운 태그를 추가하는 모달 열기`

### Step 4: 태그 추가/수정 모달 - 정보 입력
- **title**: 태그 정보 입력
- **text**: 모달에서 카테고리, 태그명, 색상을 입력합니다. 카테고리는 기존 항목 선택 또는 새로 입력 가능합니다. 색상은 8가지 프리셋에서 선택하거나 직접 지정할 수 있습니다.
- **highlight**: `#modal`
- **trigger**: none
- **tips**:
  - 카테고리는 자동완성 지원 (기존 카테고리 목록)
  - 새 카테고리도 자유롭게 입력 가능
  - 태그명은 필수 입력 항목
  - 기본 색상은 파란색(blue)
  - 색상 선택 후 호버시 테두리로 확인 가능
- **buttons**:
  - `#tagCategory / 카테고리 / 기존 카테고리 선택 또는 새로 입력`
  - `#tagName / 태그명 / 예: VIP, 관심고객`
  - `input[name="color"] / 색상 선택 / 8가지 색상 라디오 버튼`

### Step 5: 태그 저장 또는 취소
- **title**: 모달 저장/취소
- **text**: 모든 정보를 입력한 후 "저장하기" 버튼을 클릭하면 새 태그가 저장됩니다. 작업을 취소하려면 "취소" 버튼을 클릭하세요.
- **highlight**: `.flex.gap-3.pt-4`
- **trigger**: click
- **tips**:
  - 저장 후 페이지가 자동 새로고침되어 새 태그 확인 가능
  - 취소 시 입력된 정보는 버려짐
  - 성공 메시지는 토스트로 표시됨
- **buttons**:
  - `#cancelBtn / 취소 / 모달 닫기 및 변경 사항 취소`
  - `#submitBtn / 저장하기 / 새 태그 저장 또는 기존 태그 수정`

### Step 6: 태그 수정 및 삭제
- **title**: 태그 수정 및 삭제
- **text**: 태그에 마우스를 올리면 오른쪽에 삭제 아이콘이 나타납니다. 태그를 클릭하면 수정 모달이 열려 기존 정보를 변경할 수 있습니다. 삭제 아이콘을 클릭하면 태그가 제거됩니다.
- **highlight**: `.tag-item`
- **trigger**: click
- **tips**:
  - 태그 카드를 클릭하면 수정 모달 열림
  - 삭제 아이콘은 호버시만 보임 (실수 방지)
  - 삭제 전 확인 대화창 표시
  - 삭제 후 관련된 환자 태그는 자동 제거
- **buttons**:
  - `.tag-item / 태그 클릭 / 수정 모달 열기`
  - `.delete-btn / 삭제 / 태그 삭제 확인 후 제거`
