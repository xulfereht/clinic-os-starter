# 미디어 라이브러리 (/admin/media) — 투어 시나리오

## 페이지 목적
병원에서 사용하는 모든 이미지를 중앙에서 관리하고 URL을 쉽게 복사할 수 있습니다.

## 시나리오 (6 스텝)

### Step 1: 미디어 라이브러리 진입
- **title**: 미디어 관리 센터 이해하기
- **text**: 이 페이지는 병원의 모든 이미지 파일을 한 곳에서 관리하는 중앙 저장소입니다. 페이지는 3개 섹션으로 구성됩니다. 맨 위는 새 이미지를 업로드하는 영역이고, 그 아래는 검색과 필터 영역, 맨 아래는 업로드된 이미지들의 갤러리입니다. 각 이미지는 썸네일로 표시되고, 정확한 파일명과 크기 정보가 함께 표시됩니다.
- **highlight**: `.max-w-4xl` (전체 컨테이너)
- **trigger**: click
- **tips**:
  - 페이지 로드 시 전체 이미지 갤러리가 표시됩니다
  - 로딩 중에는 "Loading..." 메시지가 표시됩니다
  - 갤러리는 반응형으로 화면 크기에 맞춰 조정됩니다
- **buttons**:
  - selector: `#refresh-btn`
  - label: 새로고침
  - desc: 이미지 갤러리를 새로고침합니다

### Step 2: 새 이미지 업로드
- **title**: 이미지 파일 업로드하기
- **text**: 페이지 상단의 '새 이미지 업로드' 섹션에서 새 이미지를 업로드할 수 있습니다. 파일 선택 영역을 클릭하거나 드래그 앤 드롭으로 이미지 파일을 선택합니다. JPG, PNG, GIF 등의 일반적인 이미지 포맷을 지원합니다. 파일을 선택하면 자동으로 업로드되어 갤러리에 추가됩니다. 업로드된 파일은 즉시 다른 페이지에서 사용할 수 있습니다.
- **highlight**: `#upload-section`
- **trigger**: click
- **tips**:
  - 드래그 앤 드롭으로 편리하게 업로드할 수 있습니다
  - 여러 파일을 한 번에 업로드할 수 있습니다
  - 업로드 중에는 진행률이 표시됩니다
  - 대용량 파일은 업로드에 시간이 걸릴 수 있습니다
- **buttons**:
  - selector: `input[type="file"]`
  - label: 파일 선택
  - desc: 컴퓨터에서 이미지 파일을 선택합니다

### Step 3: 파일명으로 검색
- **title**: 특정 이미지 빠르게 찾기
- **text**: 검색과 필터 섹션의 검색창에 이미지 파일명을 입력하면 갤러리가 실시간으로 필터링됩니다. 예를 들어 "doctor"를 검색하면 파일명에 "doctor"가 포함된 이미지들만 표시됩니다. 검색은 대소문자를 구분하지 않으며, 부분 일치를 지원합니다. 검색창을 비우면 모든 이미지가 다시 표시됩니다.
- **highlight**: `#search-input`
- **trigger**: input
- **tips**:
  - 검색은 파일명으로만 이루어집니다
  - 실시간 검색이므로 즉시 결과가 반영됩니다
  - 검색창을 비우면 필터가 해제됩니다
- **buttons**: 없음 (자동 필터링)

### Step 4: 이미지 갤러리 보기
- **title**: 업로드된 이미지 확인하기
- **text**: 메인 갤러리 섹션에는 업로드된 모든 이미지가 카드 형태로 표시됩니다. 각 카드에는 이미지의 썸네일, 파일명(작은 텍스트), 파일 크기가 표시됩니다. 갤러리는 반응형으로, 화면이 좁으면 2열, 중간 크기는 3열, 넓은 화면은 4열로 표시됩니다. 이미지가 없으면 "업로드된 이미지가 없습니다" 메시지가 표시됩니다.
- **highlight**: `#gallery`
- **trigger**: hover
- **tips**:
  - 썸네일은 정사각형 비율로 자동 크롭됩니다
  - 파일명이 길면 말 줄임표(...)로 표시됩니다
  - 파일 크기는 B, KB, MB 단위로 표시됩니다
- **buttons**: 없음 (정보 표시만)

### Step 5: 이미지 URL 복사
- **title**: 이미지 URL을 다른 페이지에 사용하기
- **text**: 각 이미지 카드의 하단에는 2개의 버튼이 있습니다. 파란색 'URL 복사' 버튼을 클릭하면 해당 이미지의 전체 URL이 클립보드에 자동으로 복사됩니다. 이 URL을 이용해서 홈페이지, 블로그, 게시물 등 병원의 다른 페이지에서 이미지를 사용할 수 있습니다. URL은 한 번의 클릭으로 복사되므로 매우 편리합니다.
- **highlight**: `.flex.gap-2` (버튼 영역)
- **trigger**: click
- **tips**:
  - URL 복사 후 팝업 알림이 나타납니다
  - 복사된 URL은 절대 경로(전체 주소)입니다
  - URL은 영구적으로 유지되므로 나중에도 사용 가능합니다
  - 이미지 삭제 시 URL도 더 이상 작동하지 않습니다
- **buttons**:
  - selector: `button[onclick*="copyURL"]`
  - label: URL 복사
  - desc: 이미지 URL을 클립보드에 복사합니다

### Step 6: 이미지 삭제
- **title**: 불필요한 이미지 제거하기
- **text**: 각 이미지 카드의 오른쪽 아래에는 빨간색 '삭제' 버튼이 있습니다. 이 버튼을 클릭하면 해당 이미지를 삭제할 수 있습니다. 삭제 전에 확인 팝업이 표시되므로 실수로 삭제하는 것을 방지할 수 있습니다. 삭제된 이미지는 갤러리에서 즉시 제거되고, 해당 URL도 더 이상 작동하지 않으므로 주의해야 합니다.
- **highlight**: `button[onclick*="deleteFile"]`
- **trigger**: click
- **tips**:
  - 삭제 전에 반드시 확인창이 표시됩니다
  - 삭제된 이미지는 복구할 수 없습니다
  - 삭제 후 갤러리가 자동으로 새로고침됩니다
  - 다른 페이지에서 사용 중인 이미지를 삭제하면 그 페이지에서 이미지가 깨집니다
- **buttons**:
  - selector: `button[onclick*="deleteFile"]`
  - label: 삭제
  - desc: 선택한 이미지를 삭제합니다
