# 예약 관리 (/admin/reservations) — 투어 시나리오

## 페이지 목적
환자 예약을 등록/변경/취소하고, 내원 확인을 기록하는 페이지.

## 시나리오 (10 스텝)

### Step 1: 페이지 소개
- **title**: 예약 관리
- **text**: 환자의 예약을 관리하는 페이지입니다. 오늘 날짜의 예약이 시간순으로 표시되며, 예약 등록/상태 변경/삭제를 할 수 있습니다.
- **highlight**: 없음 (전체 페이지)
- **trigger**: confirm
- **tips**:
  - 환자가 홈페이지에서 직접 예약하면 자동으로 이 목록에 추가됩니다
  - "▼ 다음 예약 ▼" 표시로 현재 시점 기준 다음 예약을 한눈에 파악합니다
  - 지나간 시간의 예약은 흐리게 표시됩니다

### Step 2: 날짜 필터
- **title**: 예약 관리 — 날짜 선택
- **text**: 이 날짜 선택기를 클릭해서 특정 날짜의 예약을 확인해보세요. 기본값은 오늘입니다.
- **highlight**: `#filter-date`
- **trigger**: confirm
- **tips**:
  - 달력에서 날짜를 클릭하면 즉시 해당 날짜의 예약 목록이 로드됩니다

### Step 3: 담당의 필터
- **title**: 예약 관리 — 담당의 필터
- **text**: 이 드롭다운에서 특정 의료진을 선택하면, 해당 의료진의 예약만 필터링됩니다. "전체"를 선택하면 모든 예약이 표시됩니다.
- **highlight**: `#filter-doctor`
- **trigger**: confirm

### Step 4: 상태 필터
- **title**: 예약 관리 — 상태 필터
- **text**: 예약 상태로 필터링합니다. "예약됨"은 아직 방문 전, "진료 완료"는 내원 확인된 건, "노쇼"는 미방문, "취소"는 취소된 건입니다.
- **highlight**: `#filter-status`
- **trigger**: confirm
- **tips**:
  - 예약됨(파랑), 진료완료(초록), 취소(빨강), 노쇼(회색) 색상으로 구분됩니다

### Step 5: 예약 등록 버튼
- **title**: 예약 관리 — 예약 등록
- **text**: 이 버튼을 클릭하면 새 예약을 등록하는 팝업이 열립니다. 전화나 방문으로 예약을 받았을 때 사용합니다.
- **highlight**: `#add-reservation-btn`
- **trigger**: confirm

### Step 6: 예약 모달 — 환자 검색
- **title**: 예약 등록 — 환자 찾기
- **text**: 환자 이름 또는 전화번호를 입력하면 자동완성 목록이 나타납니다. 목록에서 환자를 클릭하여 선택합니다. 이름, 나이, 성별, 생년월일이 함께 표시됩니다.
- **highlight**: `#patient-search`
- **trigger**: confirm
- **tips**:
  - 이미 등록된 환자만 검색됩니다
  - 신규 환자는 먼저 "환자 관리"에서 등록 후 예약하세요

### Step 7: 예약 모달 — 날짜/시간 선택
- **title**: 예약 등록 — 날짜와 시간
- **text**: 날짜를 선택하면 아래에 예약 가능한 시간 슬롯이 표시됩니다. 초록색은 예약 가능, 노란색은 잔여 1건, 회색은 마감입니다. 원하는 시간을 클릭하세요.
- **highlight**: `#slot-picker`
- **trigger**: confirm
- **tips**:
  - 담당의를 먼저 선택하면 해당 의료진의 가용 시간만 표시됩니다
  - 시간 슬롯은 설정 > 근무/휴일에서 설정한 진료시간 기준입니다

### Step 8: 예약 모달 — 담당의/메모/저장
- **title**: 예약 등록 — 담당의와 메모
- **text**: 담당의를 지정하고 (선택사항), 방문 사유나 특이사항을 메모에 기록합니다. 모든 입력이 끝나면 "저장" 버튼을 클릭합니다.
- **highlight**: `#res-doctor`
- **trigger**: confirm
- **buttons**:
  - selector: `#res-doctor` / label: 담당의 / desc: 예약 담당 의료진을 선택합니다 (미지정 가능)
  - selector: `#res-notes` / label: 메모 / desc: 방문 사유, 특이사항을 기록합니다
  - selector: `#close-modal-btn` / label: 취소 / desc: 입력을 취소하고 모달을 닫습니다
  - selector: `button[type="submit"]` / label: 저장 / desc: 예약을 등록합니다

### Step 9: 예약 상태 변경
- **title**: 예약 관리 — 상태 변경 버튼
- **text**: 각 예약 행의 오른쪽에 상태 변경 버튼이 있습니다. 환자가 도착하면 "내원확인", 예약 시간이 지났는데 미방문이면 "노쇼", 환자가 취소하면 "취소"를 클릭합니다.
- **highlight**: 없음 (테이블 영역)
- **trigger**: confirm
- **tips**:
  - "내원확인" 클릭 → 접수 현황에 자동 반영됩니다
  - "노쇼" 기록은 환자 차트에 누적되어 관리됩니다
  - 상태 변경 시 확인 팝업이 표시됩니다 (실수 방지)
- **buttons**:
  - label: 내원확인 / desc: 환자가 도착했음을 기록합니다. 초록 뱃지로 변경됩니다
  - label: 노쇼 / desc: 예약 시간에 미방문을 기록합니다. 회색 뱃지로 변경됩니다
  - label: 취소 / desc: 예약 취소를 기록합니다. 빨간 뱃지로 변경됩니다

### Step 10: 부가 기능
- **title**: 예약 관리 — 차트/삭제
- **text**: 각 행에서 "차트" 버튼으로 해당 환자의 상세 차트 페이지로 바로 이동할 수 있습니다. "삭제"는 예약을 완전히 삭제합니다 (복구 불가).
- **highlight**: 없음
- **trigger**: confirm
- **tips**:
  - 환자 이름을 직접 클릭해도 차트 페이지로 이동합니다
  - ↻ 새로고침 버튼으로 목록을 수동 갱신할 수 있습니다
- **buttons**:
  - selector: `#refresh-btn` / label: ↻ 새로고침 / desc: 예약 목록을 다시 불러옵니다
