# 공지사항 (/admin/notices) — 투어 시나리오

## 페이지 목적
병원의 소식, 안내, 공지사항을 작성하고 관리하는 게시판입니다.

## 시나리오 (7 스텝)

### Step 1: 공지사항 목록 진입
- **title**: 공지사항 관리 시작하기
- **text**: 이 페이지에서는 병원의 모든 공지사항을 한눈에 확인하고 관리할 수 있습니다. 작성한 공지사항의 제목, 작성자, 공개 상태, 조회수, 작성일 등을 테이블 형식으로 볼 수 있습니다. 공지된 항목(상단 고정)은 빨간 '공지' 배지로 표시되며, 공개 여부에 따라 초록색 또는 회색 상태 배지가 붙습니다.
- **highlight**: `.bg-white.rounded-xl.border.border-slate-200.shadow-sm` (테이블 전체)
- **trigger**: click
- **tips**:
  - 테이블에서 한 행을 클릭하면 상세 페이지로 이동합니다
  - 상단 고정(is_pinned=1) 공지는 항상 목록 최상단에 나타납니다
  - 최신 작성 순서로 정렬되어 새 공지가 위에 옵니다
- **buttons**:
  - selector: `a[href="/admin/notices/new"]`
  - label: 공지사항 작성
  - desc: 새 공지사항을 작성합니다

### Step 2: 상태 필터 적용
- **title**: 공개/비공개 상태로 필터링
- **text**: 필터 바의 버튼들을 이용해 공지사항을 필터링할 수 있습니다. '전체' 버튼은 모든 공지를 보고, '공개' 버튼은 현재 발행된(published) 공지만, '비공개' 버튼은 아직 초안(draft) 상태인 공지만 표시합니다. 공개 공지는 환자가 홈페이지의 공지 페이지에서 볼 수 있고, 비공개는 내부용입니다.
- **highlight**: `.flex.items-center.gap-2.w-full.md:w-auto` (필터 버튼 영역)
- **trigger**: click
- **tips**:
  - 현재 활성 필터는 배경색으로 구분됩니다
  - 공개(초록색)는 published 상태입니다
  - 비공개(진회색)는 draft 상태입니다
  - 필터를 적용해도 페이지 아래 총 개수는 전체 기준입니다
- **buttons**:
  - selector: `a[href="/admin/notices?status=published"]`
  - label: 공개
  - desc: 발행된 공지사항만 표시합니다

### Step 3: 키워드 검색
- **title**: 공지사항 키워드로 찾기
- **text**: 오른쪽 검색 바에 원하는 키워드를 입력하면 제목이나 내용에 해당 단어가 있는 공지사항을 찾을 수 있습니다. 예를 들어 "휴무"를 검색하면 휴무 관련 공지만 필터링됩니다. 검색은 대소문자를 구분하지 않으며, 부분 일치도 지원합니다.
- **highlight**: `form` (검색 바 영역)
- **trigger**: input
- **tips**:
  - 검색창에 텍스트를 입력하고 '검색' 버튼을 누르면 검색됩니다
  - 빈 검색창으로 검색하면 필터가 해제됩니다
  - 상태 필터와 검색을 조합해 사용할 수 있습니다
- **buttons**:
  - selector: `button[type="submit"]`
  - label: 검색
  - desc: 입력한 키워드로 공지사항을 검색합니다

### Step 4: 공지사항 상세 열기
- **title**: 공지사항 상세 보기
- **text**: 테이블의 제목 부분을 클릭하거나 행 전체를 클릭하면 그 공지사항의 상세 페이지로 이동합니다. 여기서 전체 내용을 확인하고 수정, 삭제, 상태 변경 등의 관리 작업을 할 수 있습니다. 공지사항이 현재 공개 상태인지 비공개 상태인지도 바로 확인할 수 있습니다.
- **highlight**: `a[href*="/admin/notices/"]`
- **trigger**: click
- **tips**:
  - 제목을 클릭해도, 행 전체를 클릭해도 같은 페이지로 이동합니다
  - 상세 페이지에서는 전체 내용을 보거나 편집할 수 있습니다
  - 작성자와 작성일도 상세 페이지에서 확인됩니다
- **buttons**:
  - selector: `a[class*="group-hover:text-blue"]`
  - label: 제목 링크
  - desc: 공지사항 상세 페이지로 이동합니다

### Step 5: 페이지네이션
- **title**: 많은 공지사항 탐색하기
- **text**: 공지사항이 많으면 페이지를 나누어 표시합니다. 테이블 하단의 페이지 번호 버튼으로 다른 페이지를 볼 수 있습니다. 현재 페이지는 어두운 배경으로 표시되고, 이전/다음 버튼으로 인접한 페이지로 빠르게 이동할 수 있습니다. 각 페이지당 15개씩 표시됩니다.
- **highlight**: `.px-6.py-4.border-t.border-slate-100.flex.items-center.justify-between`
- **trigger**: click
- **tips**:
  - 한 페이지당 15개 공지사항이 표시됩니다
  - 총 개수와 현재 페이지의 범위가 표시됩니다
  - 페이지 이동 시 검색/필터 조건이 유지됩니다
- **buttons**:
  - selector: `a[href*="page="]`
  - label: 페이지 번호
  - desc: 해당 페이지의 공지사항을 표시합니다

### Step 6: 조회수 확인
- **title**: 공지사항 인기도 파악하기
- **text**: 테이블의 '조회수' 열은 각 공지사항이 환자에게 몇 번 조회되었는지를 나타냅니다. 높은 조회수는 그 공지사항이 많은 환자들에게 관심을 받았다는 의미입니다. 조회수가 0이면 아직 누구도 클릭하지 않았다는 뜻입니다. 중요한 공지일수록 조회수가 높은 경향이 있습니다.
- **highlight**: `td:nth-child(5)` (조회수 칼럼)
- **trigger**: hover
- **tips**:
  - 조회수는 공개된 공지만 증가합니다
  - 비공개(draft) 공지는 항상 0입니다
  - 조회수는 자동으로 집계됩니다
- **buttons**: 없음 (정보 열람만)

### Step 7: 새 공지사항 작성
- **title**: 새로운 공지사항 등록하기
- **text**: 우측 상단의 파란 '공지사항 작성' 버튼을 클릭하면 새 공지사항 작성 페이지로 이동합니다. 제목, 내용, 공개 여부, 고정 여부 등을 설정할 수 있습니다. 작성한 공지는 먼저 비공개(draft)로 저장되므로, 검토 후에 공개 상태로 변경할 수 있습니다.
- **highlight**: `a[href="/admin/notices/new"]`
- **trigger**: click
- **tips**:
  - 새 공지사항은 항상 비공개(draft)로 시작됩니다
  - 작성하면서 임시 저장할 수 있습니다
  - 작성 완료 후 미리보기로 환자 입장에서 확인할 수 있습니다
  - 공지 버튼을 켜면 목록 최상단에 고정됩니다
- **buttons**:
  - selector: `button[type="submit"]`
  - label: 발행
  - desc: 작성한 공지사항을 저장하고 공개합니다
