# Promotions (/admin/settings/promotions) — 투어 시나리오

## 페이지 목적
결제 시 적용할 할인 규칙을 관리하며, 정률(%) 할인과 정액(원) 할인을 유연하게 설정합니다.

## 시나리오 (5 스텝)

### Step 1: 프로모션 목록 조회
- **title**: 프로모션 관리 페이지 개요
- **text**: 이 페이지에서 등록된 모든 프로모션을 테이블 형식으로 볼 수 있습니다. 프로모션명, 할인 유형, 할인값, 활성화 상태가 표시됩니다.
- **highlight**: `.bg-white.rounded-2xl.shadow-sm`
- **trigger**: none
- **tips**:
  - 테이블 행은 호버시 배경색이 변함
  - 상태 배지로 활성/비활성 한눈에 구분
  - 정률(%)은 보라색, 정액(원)은 주황색 배지로 표시
- **buttons**: []

### Step 2: 새 프로모션 추가
- **title**: 새 프로모션 추가 버튼
- **text**: 우측 상단의 "+ 프로모션 추가" 파란색 버튼을 클릭하면 새 프로모션 추가 모달이 열립니다.
- **highlight**: `#addPromoBtn`
- **trigger**: click
- **tips**:
  - 버튼은 항상 테이블 상단에 고정되어 있음
  - 클릭하면 비어있는 모달 양식이 열림
- **buttons**:
  - `#addPromoBtn / 프로모션 추가 / 새 프로모션 추가 모달 열기`

### Step 3: 프로모션 정보 입력
- **title**: 프로모션 정보 입력
- **text**: 모달에서 프로모션명, 할인 유형(정률/정액), 할인값을 입력합니다. 할인 유형을 선택하면 할인값 입력 방식이 안내됩니다. 정률인 경우 20을 입력하면 20% 할인을 의미합니다.
- **highlight**: `#modal`
- **trigger**: none
- **tips**:
  - 프로모션명은 필수 입력
  - 할인값도 필수 입력 (숫자만)
  - 정액 할인: 원 단위 입력 (예: 10000 = 10,000원)
  - 정률 할인: 백분율 입력 (예: 20 = 20%)
  - 할인값은 음수 불가능
- **buttons**:
  - `#promoName / 프로모션명 / 예: 여름 정기 세일`
  - `#promoType / 할인 유형 / 정률(%) 또는 정액(원) 선택`
  - `#promoValue / 할인값 / 숫자만 입력`

### Step 4: 상태 설정 (수정 시만 표시)
- **title**: 프로모션 상태 설정
- **text**: 기존 프로모션을 수정할 때만 "상태" 필드가 표시됩니다. 프로모션을 활성화하거나 비활성화할 수 있습니다. 비활성 프로모션은 결제 시 적용되지 않습니다.
- **highlight**: `#statusField`
- **trigger**: none
- **tips**:
  - 새 프로모션 추가 시에는 상태 필드가 숨겨짐 (자동 활성화)
  - 수정 모달 열때만 상태 선택 가능
  - 활성/비활성 토글로 빠른 관리 가능
- **buttons**:
  - `#promoStatus / 상태 선택 / 활성 또는 비활성`

### Step 5: 저장, 수정, 삭제
- **title**: 프로모션 저장 및 관리
- **text**: 추가 버튼을 클릭하면 "저장하기" 버튼으로 표시되고, 기존 프로모션의 수정 버튼을 클릭하면 "수정하기" 버튼으로 표시됩니다. 테이블의 삭제 버튼으로 프로모션을 제거할 수 있습니다.
- **highlight**: `.flex.gap-3.pt-4`
- **trigger**: click
- **tips**:
  - 저장/수정 후 페이지가 자동 새로고침됨
  - 삭제 전 확인 대화창이 표시됨
  - 성공 메시지는 토스트로 표시
  - 한 번에 하나의 프로모션만 편집 가능
- **buttons**:
  - `#cancelBtn / 취소 / 모달 닫기 및 변경 사항 취소`
  - `#submitBtn / 추가하기 또는 수정하기 / 프로모션 저장 또는 업데이트`
  - `.edit-btn / 수정 / 기존 프로모션 수정 모달 열기`
  - `.delete-btn / 삭제 / 프로모션 삭제 확인 후 제거`
