# 플러그인 관리 (/admin/plugins) — 투어 시나리오

## 페이지 목적
설치된 플러그인을 조회하고, 활성화/비활성화를 관리하며, 플러그인 스토어에 접근하는 페이지

## 시나리오 (5 스텝)

### Step 1: 플러그인 관리 페이지 진입
- **title**: 플러그인 관리 페이지 개요
- **text**: 이 페이지에서는 한의원 사이트에 설치된 모든 플러그인을 관리할 수 있습니다. 상단에는 페이지 제목과 설명이 있으며, 아래로 설치된 플러그인 목록이 표시됩니다. 각 플러그인은 카드 형태로 표시되며, 활성화/비활성화 토글, 설정 버튼, 삭제 옵션 등을 제공합니다.
- **highlight**: `.max-w-6xl.mx-auto`
- **trigger**: confirm
- **tips**:
  - 플러그인은 카테고리별로 그룹화될 수 있음
  - 기본 플러그인(코어 플러그인)은 비활성화 불가능
  - 활성화된 플러그인은 사이트에 즉시 반영됨

### Step 2: 플러그인 카드 구성
- **title**: 플러그인 정보 카드
- **text**: 각 플러그인 카드에는 플러그인 아이콘/썸네일, 이름, 간단한 설명, 버전 정보, 작성자, 설명이 표시됩니다. 카드의 우측 상단에는 활성화/비활성화 토글 버튼이 있으며, 아래에는 설정, 상세정보, 삭제 등의 액션 버튼들이 배치되어 있습니다.
- **highlight**: `.grid.gap-6 > div`
- **trigger**: confirm
- **tips**:
  - 카드 배경은 활성 상태를 시각적으로 표현함 (활성=밝음, 비활성=어두움)
  - 플러그인 이름은 클릭 시 상세정보 페이지로 이동
  - 아이콘 없는 플러그인은 기본 아이콘으로 표시됨

### Step 3: 플러그인 활성화/비활성화 토글
- **title**: 플러그인 토글 제어
- **text**: 각 플러그인 카드의 우측 상단에 있는 토글 버튼(활성화/비활성화)을 클릭하여 플러그인을 활성/비활성 상태로 변경할 수 있습니다. 토글 색상과 텍스트가 상태에 따라 변경되며, 저장 중 임시로 로딩 표시가 나타납니다. 변경 후 사이트가 자동으로 업데이트됩니다.
- **highlight**: `[role="switch"]`
- **trigger**: click
- **tips**:
  - 토글 작동 시 API 요청이 백그라운드에서 처리됨
  - 토글 중에는 다른 작업을 수행할 수 없음
  - 플러그인 비활성화 시 관련 데이터는 보존됨
  - 토글 실패 시 토스트 알림으로 오류 메시지 표시

### Step 4: 플러그인 액션 버튼
- **title**: 플러그인 설정 및 상세정보
- **text**: 각 플러그인 카드 하단에는 여러 액션 버튼들이 배치되어 있습니다. '⚙️ 설정' 버튼은 플러그인의 고급 설정 페이지로 이동하며, '📖 상세정보' 버튼은 플러그인의 설명서 및 가이드를 표시합니다. '🗑️ 삭제' 버튼은 플러그인을 제거하는데, 이 버튼은 기본 플러그인에는 표시되지 않습니다.
- **highlight**: `button:has-text("⚙️"), button:has-text("📖"), button:has-text("🗑️")`
- **trigger**: click
- **buttons**:
  - selector: `button:has-text("⚙️")`
  - label: "⚙️ 설정"
  - desc: 플러그인 설정 페이지로 이동
  - selector: `button:has-text("📖")`
  - label: "📖 상세정보"
  - desc: 플러그인 설명서 표시
  - selector: `button:has-text("🗑️")`
  - label: "🗑️ 삭제"
  - desc: 플러그인 제거
- **tips**:
  - 설정 버튼은 플러그인이 설정을 지원할 때만 활성화됨
  - 상세정보는 사이드 패널이나 모달로 표시될 수 있음
  - 삭제 시 확인 대화창이 나타남

### Step 5: 플러그인 스토어 링크
- **title**: 새 플러그인 설치 (플러그인 스토어)
- **text**: 페이지 상단 또는 하단에 '플러그인 스토어' 또는 '+ 새 플러그인 설치' 링크가 있습니다. 이를 클릭하면 Clinic-OS 플러그인 마켓플레이스(외부 링크)로 이동하여 추가 플러그인을 검색하고 설치할 수 있습니다.
- **highlight**: `a:has-text("플러그인 스토어"), button:has-text("+ 새 플러그인")`
- **trigger**: click
- **tips**:
  - 플러그인 스토어 링크는 새 탭에서 열림
  - 스토어에서 설치한 플러그인은 자동으로 목록에 추가됨
  - 평가/리뷰를 통해 신뢰할 수 있는 플러그인 선택 가능
