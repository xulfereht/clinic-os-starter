# Chat Widget Settings (/admin/settings/widget) — 투어 시나리오

## 페이지 목적
웹사이트 우측/좌측 하단에 표시되는 고객 상담 채팅 위젯의 외관, 메시지, 입력 필드를 통합 설정합니다.

## 시나리오 (8 스텝)

### Step 1: 위젯 활성화 및 위치 선택
- **title**: 채팅 위젯 기본 설정
- **text**: "위젯 활성화" 토글을 켜면 웹사이트에 채팅 위젯이 표시됩니다. "위젯 위치"에서 오른쪽 하단 또는 왼쪽 하단 중 선택하여 배치를 결정합니다.
- **highlight**: `.grid.grid-cols-1.md\\:grid-cols-2.gap-6`
- **trigger**: none
- **tips**:
  - 토글 ON=웹사이트에 위젯 표시, OFF=숨김
  - 위치 선택은 즉시 반영 (저장 필수)
  - 모바일과 데스크톱 모두에 적용
- **buttons**:
  - `#enabled / 위젯 활성화 토글 / 채팅 위젯 표시/숨김`
  - `#position / 위젯 위치 선택 / 오른쪽 또는 왼쪽 하단`

### Step 2: 디자인 - 주요 색상 설정
- **title**: 위젯 색상 커스터마이징
- **text**: "주요 색상" 섹션에서 위젯의 기본 색상을 설정합니다. 색상 피커(좌측 원형)를 클릭하거나 HEX 코드(우측 입력창)를 직접 입력할 수 있습니다. 아래 색상 프리셋(파란색, 초록색, 보라색 등)을 클릭하면 빠르게 변경 가능합니다.
- **highlight**: `.grid.grid-cols-1.md\\:grid-cols-2.gap-6`
- **trigger**: none
- **tips**:
  - 피커와 입력창은 양방향 동기화
  - 유효한 HEX 코드만 반영 (#RRGGBB 형식)
  - 5개 색상 프리셋으로 신속한 선택
  - 한번 선택하면 모든 위젯 UI에 적용
- **buttons**:
  - `#primaryColor / 색상 피커 / 시각적으로 색상 선택`
  - `#primaryColorText / HEX 코드 입력 / 정확한 색상값 입력 (예: #667eea)`
  - `.color-preset / 색상 프리셋 / 사전 정의된 색상 빠른 선택`

### Step 3: 공식 프로필 설정
- **title**: 상담원 프로필 (고객 노출 정보)
- **text**: "공식 프로필" 섹션에서 고객에게 보여지는 상담원 이름과 프로필 이미지를 설정합니다. 입력한 이름은 모든 위젯 메시지 위에 통일되어 표시되며, 실제 답변자는 관리자 메신저에만 기록됩니다.
- **highlight**: `.bg-white.rounded-2xl.shadow-sm.border.border-slate-200.p-6`
- **trigger**: none
- **tips**:
  - 프로필 이름은 필수 입력 (기본: "Customer Support")
  - 프로필 이미지는 선택사항 (비우면 기본 아이콘)
  - 이미지는 정사각형(200x200px 이상) 권장
  - 관리자에게는 실제 직원 정보가 항상 표시됨 (투명성)
- **buttons**:
  - `#officialName / 프로필 이름 / 고객에게 보여지는 이름`
  - `#officialAvatar / 프로필 이미지 URL / 선택 (비우면 기본 아이콘)`

### Step 4: 환영 메시지 및 오프라인 메시지
- **title**: 채팅 메시지 설정
- **text**: "환영 메시지"는 위젯 열 때 가장 먼저 표시되는 인사말입니다. "오프라인 메시지"는 운영시간 외에 사용자가 메시지를 보낼 때 표시되는 응답입니다. 두 메시지 모두 고객에게 친근하고 명확하게 작성하세요.
- **highlight**: `.space-y-4`
- **trigger**: none
- **tips**:
  - 환영 메시지: 50-100글자 권장
  - 오프라인 메시지: 운영시간 안내 포함 권장
  - 한글, 영문, 이모지 모두 지원
  - 실시간 미리보기 지원 (저장 후 확인 필요)
- **buttons**:
  - `#welcomeMessage / 환영 메시지 / 위젯 열 때 표시되는 첫 메시지`
  - `#offlineMessage / 오프라인 메시지 / 운영시간 외 표시 메시지`

### Step 5: 자동 응답 메시지 활성화
- **title**: 자동 응답 메시지
- **text**: "자동 응답 메시지 활성화" 토글을 켜면 고객이 메시지를 보낼 때 즉시 응답합니다. 토글을 켜면 아래에 자동 응답 텍스트 입력 필드가 나타나 메시지 내용을 지정할 수 있습니다.
- **highlight**: `#autoReplySection`
- **trigger**: none
- **tips**:
  - 토글 ON=자동 응답 활성화
  - OFF 시 자동 응답 필드 숨김
  - 자동 응답은 운영시간 관계없이 항상 작동
  - 실제 상담원의 답변은 별도로 가능
- **buttons**:
  - `#autoReply / 자동 응답 활성화 / 토글 ON/OFF`
  - `#autoReplyMessage / 자동 응답 메시지 / 고객에게 보낼 자동 메시지`

### Step 6: 운영 시간 표시
- **title**: 운영 시간 표시 설정
- **text**: "운영 시간 표시" 토글을 켜면 위젯에 운영 시간이 표시됩니다. 기본값은 "근무/휴일 관리" 페이지에서 설정한 시간이 자동으로 적용되지만, 직접 입력하면 해당 값으로 덮어씁니다. 비우면 근무 일정에서 자동으로 가져옵니다.
- **highlight**: `.space-y-4`
- **trigger**: none
- **tips**:
  - 자동 연동: 근무/휴일 관리 페이지 설정 자동 적용
  - 직접 입력: "평일 09:00 - 18:00 | 토 09:00 - 13:00" 형식
  - 비우면 자동값 사용
  - 고객은 운영시간 외 오프라인 메시지 받음
- **buttons**:
  - `#showBusinessHours / 운영 시간 표시 / 토글 ON/OFF`
  - `#businessHours / 운영 시간 직접 입력 / 자동 연동 덮어쓰기 (선택)`

### Step 7: 입력 필드 설정
- **title**: 고객 정보 수집 필드
- **text**: "이름 입력 필드 표시"와 "연락처 입력 필드 표시" 체크박스로 고객이 메시지를 보낼 때 입력해야 할 정보를 지정합니다. 연락처는 필수이며, 이름은 선택사항입니다.
- **highlight**: `.space-y-4`
- **trigger**: none
- **tips**:
  - 이름: 선택사항, 체크 해제 가능
  - 연락처: 필수항목 (항상 수집)
  - 수집된 정보는 리드로 자동 저장
  - GDPR 등 개인정보 정책 준수
- **buttons**:
  - `#collectName / 이름 필드 표시 / 고객 이름 수집 여부`
  - `#collectPhone / 연락처 필드 표시 / 고객 전화번호 필수 수집`

### Step 8: 링크 공유 및 저장
- **title**: 위젯 공유 링크 및 설정 저장
- **text**: "링크 공유" 섹션에서 고객이 직접 채팅을 시작할 수 있는 링크를 복사하여 카카오톡, 문자 등으로 공유합니다. 우측 "설정 저장" 버튼을 클릭하여 모든 위젯 설정을 저장합니다.
- **highlight**: `.flex.justify-end`
- **trigger**: click
- **tips**:
  - 공유 링크: /chat (기본 주소)
  - 추적 파라미터: ?from=kakao, ?from=sms 등으로 출처 추적 가능
  - 링크 복사 후 자유롭게 배포 가능
  - 저장 전 변경 사항은 적용되지 않음
  - 저장 후 성공 메시지 표시
- **buttons**:
  - `#shareLink / 공유 링크 / 고객 초대 링크 복사`
  - `.복사 버튼 / 복사 / 링크를 클립보드에 복사`
  - `button[type="submit"] / 설정 저장 / 모든 위젯 설정 저장`
