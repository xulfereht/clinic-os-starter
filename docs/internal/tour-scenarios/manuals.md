# 업무 매뉴얼 (/admin/manuals) — 투어 시나리오

## 페이지 목적
병원의 업무 절차와 가이드를 문서화하고 직원이 언제든 참고할 수 있도록 관리합니다.

## 시나리오 (5 스텝)

### Step 1: 매뉴얼 홈 진입
- **title**: 업무 매뉴얼 센터 이해하기
- **text**: 이 페이지는 병원의 모든 업무 매뉴얼과 가이드가 모여 있는 문서 센터입니다. 좌측 사이드바는 카테고리별 매뉴얼 목차를 보여주고, 메인 영역에는 대시보드 통계와 최근 업데이트된 문서 목록이 표시됩니다. 3개의 통계 카드는 전체 문서 수, 카테고리 수, 최근 업데이트 건수를 한눈에 보여줍니다.
- **highlight**: `.max-w-4xl` (메인 콘텐츠)
- **trigger**: click
- **tips**:
  - 사이드바를 통해 카테고리별로 빠르게 접근할 수 있습니다
  - 통계는 매번 페이지 로드 시 자동 갱신됩니다
  - 최근 업데이트 목록은 가장 최신 6개 문서만 표시됩니다
- **buttons**: 없음 (정보 표시만)

### Step 2: 통계 카드로 매뉴얼 현황 파악
- **title**: 매뉴얼 관리 현황 확인하기
- **text**: 상단의 3개 통계 카드를 통해 현재 관리 중인 매뉴얼의 규모를 파악할 수 있습니다. '전체 문서'는 등록된 모든 매뉴얼의 총 개수, '카테고리'는 분류된 주제의 수, '최근 업데이트'는 지난 시간 동안 수정된 문서의 건수입니다. 이 숫자들이 높을수록 병원의 업무 프로세스가 잘 문서화되어 있다는 의미입니다.
- **highlight**: `.grid.grid-cols-1.md:grid-cols-3.gap-6` (통계 카드)
- **trigger**: hover
- **tips**:
  - 카테고리 수가 많으면 체계적인 분류가 이루어지고 있다는 뜻입니다
  - 최근 업데이트 건수가 높으면 매뉴얼이 현행화되고 있다는 의미입니다
  - 통계 카드를 클릭해도 추가 페이지로 이동하지 않습니다
- **buttons**: 없음 (정보 표시만)

### Step 3: 최근 업데이트 문서 확인
- **title**: 최근 수정된 매뉴얼 보기
- **text**: 메인 영역 아래의 '최근 업데이트된 문서' 섹션에서는 최근 수정된 6개 매뉴얼이 카드로 표시됩니다. 각 카드에는 카테고리 배지, 제목, 마지막 수정 담당자, 수정 날짜가 표시됩니다. 카드를 클릭하면 해당 매뉴얼의 상세 페이지로 이동해 전체 내용을 확인하고 편집할 수 있습니다.
- **highlight**: `.grid.grid-cols-1.md:grid-cols-2.gap-4`
- **trigger**: click
- **tips**:
  - 각 카드는 마우스를 올리면 약간 들어올려지는 호버 효과가 있습니다
  - 카테고리 배지의 색상은 카테고리마다 다를 수 있습니다
  - 수정 날짜는 가장 최근 수정일입니다
- **buttons**:
  - selector: `a[href*="/admin/manuals/"]`
  - label: 문서 카드
  - desc: 해당 매뉴얼의 상세 페이지로 이동합니다

### Step 4: 사이드바로 카테고리 탐색
- **title**: 좌측 사이드바로 매뉴얼 찾기
- **text**: 왼쪽 사이드바는 병원의 모든 매뉴얼이 카테고리별로 정렬되어 있습니다. 각 카테고리를 클릭하면 그 카테고리에 속한 모든 매뉴얼이 표시됩니다. 예를 들어 '접수/상담'을 클릭하면 접수와 상담 관련 매뉴얼만 볼 수 있습니다. 검색 기능도 있어서 특정 매뉴얼을 빠르게 찾을 수 있습니다.
- **highlight**: `div[class*="sidebar"]`
- **trigger**: click
- **tips**:
  - 사이드바를 스크롤해서 모든 카테고리를 확인할 수 있습니다
  - 현재 보고 있는 카테고리는 하이라이트됩니다
  - 모바일에서는 사이드바가 햄버거 메뉴로 축소될 수 있습니다
- **buttons**:
  - selector: `a[class*="category"]`
  - label: 카테고리 링크
  - desc: 해당 카테고리의 매뉴얼을 표시합니다

### Step 5: 이용 가이드 확인
- **title**: 매뉴얼 작성 및 관리 방법
- **text**: 페이지 하단의 '💡 이용 가이드' 섹션에는 매뉴얼을 어떻게 활용하는지 안내합니다. 좌측 사이드바에서 문서를 검색하고 탐색할 수 있으며, 문서 내에서 @기호를 입력해 다른 문서를 링크할 수 있습니다. 또한 이미지를 붙여넣기(Ctrl+V)해서 쉽게 업로드할 수 있어 매뉴얼 작성이 매우 간편합니다.
- **highlight**: `.mt-12.p-6.bg-slate-50.rounded-xl` (이용 가이드)
- **trigger**: hover
- **tips**:
  - @ 링크는 동일 매뉴얼 또는 다른 문서 참조에 유용합니다
  - Ctrl+V는 스크린샷을 직접 붙여넣기하는 기능으로 매우 편리합니다
  - 드래그 앤 드롭으로도 이미지를 업로드할 수 있습니다
  - 모바일에서는 붙여넣기 기능이 제한될 수 있습니다
- **buttons**:
  - selector: `li` (가이드 항목)
  - label: 가이드 항목
  - desc: 매뉴얼 작성 팁을 제공합니다
