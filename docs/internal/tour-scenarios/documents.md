# 사내 문서함 (/admin/documents) — 투어 시나리오

## 페이지 목적
중요한 병원 운영 서류(사업자등록증, 계약서, 마케팅 자료 등)를 안전하게 보관하고 관리합니다.

## 시나리오 (7 스텝)

### Step 1: 문서함 진입 및 저장 현황 확인
- **title**: 사내 문서 저장소 둘러보기
- **text**: 이 페이지는 병원의 중요한 서류를 모두 안전하게 관리하는 클라우드 드라이브입니다. 우측 상단에 현재 저장된 파일 개수와 사용 중인 용량이 표시됩니다. 왼쪽 사이드바에서는 카테고리별로 문서를 분류할 수 있고, 메인 테이블에는 업로드된 모든 파일의 목록이 보입니다.
- **highlight**: `.grid.grid-cols-1.lg:grid-cols-4.gap-8` (전체 레이아웃)
- **trigger**: click
- **tips**:
  - 사용 공간은 MB 단위로 표시됩니다
  - 파일 개수와 용량은 선택한 카테고리에 따라 변합니다
  - 모든 파일은 클라우드에 자동 백업됩니다
- **buttons**:
  - selector: `button[onclick="openUploadModal()"]`
  - label: 문서 업로드
  - desc: 새 파일을 업로드합니다

### Step 2: 카테고리별 필터링
- **title**: 카테고리로 문서 분류하기
- **text**: 왼쪽 사이드바의 카테고리 목록에서 원하는 항목을 클릭하면 해당 카테고리의 문서만 필터링됩니다. 사업자등록증, 서식/양식, 계약서, 마케팅자료, 기타 등으로 분류할 수 있습니다. '전체'를 선택하면 모든 카테고리의 파일을 다시 볼 수 있습니다. 현재 선택된 카테고리는 파란색으로 하이라이트됩니다.
- **highlight**: `.lg:col-span-1.space-y-2` (카테고리 사이드바)
- **trigger**: click
- **tips**:
  - 각 카테고리를 클릭하면 체크 마크(✓)가 표시됩니다
  - 카테고리 필터는 URL 쿼리에 반영됩니다
  - 파일의 카테고리는 업로드 시에 지정하거나 나중에 변경할 수 있습니다
- **buttons**:
  - selector: `a[href*="category="]`
  - label: 카테고리 링크
  - desc: 해당 카테고리의 문서만 표시합니다

### Step 3: 파일명으로 검색
- **title**: 특정 문서 빠르게 찾기
- **text**: 메인 테이블 위의 검색창에 파일명의 일부를 입력하면 실시간으로 해당하는 파일들만 필터링됩니다. 예를 들어 "임대"를 검색하면 파일명이나 설명에 임대라는 단어가 있는 모든 문서가 표시됩니다. 검색은 대소문자를 구분하지 않습니다.
- **highlight**: `#searchInput`
- **trigger**: input
- **tips**:
  - 검색은 실시간으로 작동합니다
  - 검색창을 비우면 필터가 해제됩니다
  - 파일명뿐만 아니라 설명도 검색 대상에 포함됩니다
- **buttons**: 없음 (자동 필터링)

### Step 4: 정렬 변경
- **title**: 파일 정렬 순서 바꾸기
- **text**: 검색창 옆의 드롭다운에서 정렬 방식을 변경할 수 있습니다. '최신순'은 가장 최근에 업로드한 파일부터 보여주고, '오래된순'은 가장 오래된 파일부터, '이름순'은 파일명의 가나다 순으로 정렬합니다. 정렬 방식을 바꿔도 이미 필터링된 파일들만 정렬됩니다.
- **highlight**: `#sortSelect`
- **trigger**: click
- **tips**:
  - 기본 정렬은 '최신순'입니다
  - 정렬 방식은 모든 파일에 동일하게 적용됩니다
- **buttons**:
  - selector: `#sortSelect option`
  - label: 정렬 옵션
  - desc: 파일 목록의 정렬 순서를 바꿉니다

### Step 5: 파일 다운로드 및 공유
- **title**: 문서 다운로드와 외부 공유
- **text**: 테이블의 각 행에 마우스를 올리면 오른쪽 끝에 3개의 버튼이 나타납니다. 첫 번째 다운로드 버튼을 클릭하면 해당 파일을 컴퓨터에 다운로드할 수 있습니다. 두 번째 링크 버튼을 클릭하면 외부 공유 URL이 클립보드에 복사되어 필요한 사람에게 파일을 공유할 수 있습니다.
- **highlight**: `td:last-child .flex.justify-end` (관리 버튼 영역)
- **trigger**: hover
- **tips**:
  - 다운로드 버튼은 자동으로 원래 파일명으로 저장됩니다
  - 외부 공유 URL은 클립보드에 복사됩니다
  - 공유 URL을 통해 누구든 파일을 접근할 수 있으므로 주의하세요
- **buttons**:
  - selector: `a[download]`
  - label: 다운로드
  - desc: 파일을 다운로드합니다
  - selector: `button[onclick*="copyLink"]`
  - label: 공유 링크 복사
  - desc: 외부 공유 URL을 복사합니다

### Step 6: 파일 정보 확인
- **title**: 파일 상세 정보 보기
- **text**: 테이블의 각 행은 파일의 이름, 분류, 크기, 업로드 일시를 보여줍니다. 파일명 아래에는 선택사항으로 입력한 설명이 회색 텍스트로 표시됩니다. 파일 크기는 KB 단위로 표시되며, 업로드 일시는 년-월-일 시:분:초 형식입니다. 이 정보들을 통해 어떤 문서가 어떤 시점에 저장되었는지 한눈에 파악할 수 있습니다.
- **highlight**: `tbody tr` (파일 행)
- **trigger**: hover
- **tips**:
  - 파일명이 긴 경우 한 줄로 표시되고 말 줄임표(...)가 붙습니다
  - 크기는 1,024KB = 1MB입니다
  - 업로드 일시는 아시아/서울 표준시 기준입니다
- **buttons**: 없음 (정보 열람만)

### Step 7: 새 파일 업로드
- **title**: 새로운 문서 업로드하기
- **text**: 우측 상단의 파란 '문서 업로드' 버튼을 클릭하면 업로드 모달이 열립니다. 파일을 드래그 앤 드롭하거나 클릭해서 파일을 선택합니다. 드롭존에 파일을 끌어다 놓으면 선택됩니다. 파일을 선택한 후 카테고리를 지정하고(필수), 원하면 설명을 추가한(선택) 후 업로드 버튼을 누르면 됩니다. 최대 50MB까지 업로드 가능합니다.
- **highlight**: `#uploadModal`
- **trigger**: click
- **tips**:
  - 드래그 앤 드롭으로 편리하게 업로드할 수 있습니다
  - 카테고리는 필수 입력입니다
  - 설명은 파일명으로는 알 수 없는 내용을 추가할 때 유용합니다
  - 업로드 중 진행률이 표시됩니다
- **buttons**:
  - selector: `#uploadForm button[type="submit"]`
  - label: 업로드
  - desc: 선택한 파일을 클라우드에 업로드합니다
  - selector: `#dropzone`
  - label: 드롭존
  - desc: 파일을 드래그 앤 드롭하거나 클릭해 선택합니다
