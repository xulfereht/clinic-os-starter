# 접수 현황 (/admin/intake) — 투어 시나리오

## 페이지 목적
일별 신규 환자 접수(초진/재진)를 확인하고, 당일 또는 특정 날짜의 접수 내역을 검색하며, 각 환자의 차트로 빠르게 이동하는 일일 접수 관리 페이지입니다.

## 시나리오 (11 스텝)

### Step 1: 페이지 진입 및 페이지 헤더 확인
- **title**: 접수 현황 페이지 진입
- **text**: 페이지에 진입하면, 상단에 제목과 설명이 표시됩니다:
  - **제목**: "접수 현황"
  - **설명**: "일별 초진/재진 접수 내역을 확인합니다."
  
  이 페이지는 매일 아침 첫 업무로 접수한 환자 목록을 확인하는 용도로 설계되었습니다. 병원 직원들이 당일 환자 동선을 파악하고, 접수 현황을 실시간으로 추적합니다.
- **highlight**: 없음
- **trigger**: confirm
- **tips**:
  - 이 페이지는 매일 새로 고침할 때마다 "오늘" 날짜로 자동 설정됩니다
  - 과거 날짜의 접수 현황을 조회할 수도 있으므로, 어느 날이든 상관없습니다
  - 모바일 환경에서도 같은 기능을 모두 사용할 수 있습니다

---

### Step 2: 통계 카드 - 선택된 날짜 접수 수 확인
- **title**: 당일 총 접수 현황 (Selected Day Stats)
- **text**: 페이지 상단의 세 개 통계 카드 중 첫 번째 카드(흰색, 기본 스타일)에는:
  - **레이블**: "📅 선택된 날짜 (2025-04-07)" - 현재 선택된 날짜가 표시됩니다
  - **큰 숫자**: "7건" - 그 날짜에 접수된 총 환자 수(초진+재진 합계)
  
  이 숫자는 실시간으로 갱신되지 않지만, 페이지를 새로고침하면 최신 수치가 반영됩니다. 날짜를 변경하면 즉시 해당 날짜의 접수 수로 업데이트됩니다.
- **highlight**: `.bg-white.p-6.rounded-2xl` (첫 번째 카드)
- **trigger**: confirm
- **tips**:
  - 이 숫자가 0이면, 그 날짜에 접수가 없다는 뜻입니다
  - 주말이나 휴무일의 접수 현황을 조회해도 0으로 표시됩니다
  - 월요일 아침에 이 숫자를 확인하면, 주말 간 포함된 환자가 있는지 알 수 있습니다

---

### Step 3: 통계 카드 - 초진 접수 수 확인
- **title**: 신규 환자 접수 현황 (First Visit Stats)
- **text**: 두 번째 카드(초록색 배경, bg-emerald-50)에는:
  - **레이블**: "🆕 초진 접수"
  - **큰 숫자**: "2건"
  
  이는 선택된 날짜에 처음 방문한 신규 환자 수입니다. 초진 접수는 한의원의 신규 리드 유입을 나타내는 중요한 지표입니다. 예를 들어 당일 총 7건 중 초진이 2건이면, 신규 고객 비율이 약 29%라는 뜻입니다.
- **highlight**: `.bg-emerald-50.p-6.rounded-2xl` (두 번째 카드)
- **trigger**: confirm
- **tips**:
  - 초진 비율이 높을수록, 마케팅 활동이 잘 작동하고 있다는 신호입니다
  - 초진 비율이 지속적으로 낮으면, 마케팅 예산이나 채널을 검토하세요
  - 월별 초진 수를 기록해두면, 월간 신규 고객 유입 추세를 파악할 수 있습니다

---

### Step 4: 통계 카드 - 재진 내원 수 확인
- **title**: 기존 환자 재방문 현황 (Return Visit Stats)
- **text**: 세 번째 카드(파란색 배경, bg-blue-50)에는:
  - **레이블**: "🔄 재진 내원"
  - **큰 숫자**: "5건"
  
  이는 선택된 날짜에 재방문한 기존 환자 수입니다. 재진 비율이 높을수록, 환자 만족도와 로열티가 높다는 의미입니다. 예를 들어 당일 총 7건 중 재진이 5건이면, 재진율이 약 71%입니다.
- **highlight**: `.bg-blue-50.p-6.rounded-2xl` (세 번째 카드)
- **trigger**: confirm
- **tips**:
  - 재진율 = 재진 / 총 접수 수로 계산됩니다
  - 재진율이 80% 이상이면, 고객 만족도가 매우 높다는 뜻입니다
  - 재진율이 50% 이하로 떨어지면, 진료 만족도나 리콜 시스템을 검토하세요
  - 초진과 재진의 비율을 함께 보면, 한의원의 성장 방향을 파악할 수 있습니다

---

### Step 5: 날짜 네비게이션 - 이전 날짜 버튼
- **title**: 이전 날짜로 이동
- **text**: 페이지 상단의 날짜 네비게이션 영역에서 맨 왼쪽에 좌측 화살표(←) 아이콘 버튼이 있습니다. 이 버튼을 클릭하면 하루 전 날짜의 접수 현황을 조회합니다. 예를 들어 현재 2025-04-07을 보고 있으면, 이 버튼을 클릭하면 2025-04-06의 데이터로 변경됩니다. 버튼 위로 마우스를 올리면 배경이 밝아져 클릭 가능함을 알립니다. 계속 클릭하여 여러 날짜 전으로 거슬러 올라갈 수 있습니다.
- **highlight**: `a[href*="?date="]` (이전 화살표 링크)
- **trigger**: click
- **tips**:
  - 이 버튼을 반복 클릭하여 지난주 접수 현황을 빠르게 확인할 수 있습니다
  - 특정 요일의 특성을 분석하고 싶으면 (예: 월요일이 피크인지), 여러 주의 같은 요일을 비교하세요
  - 매우 과거 날짜(예: 작년)의 데이터도 조회할 수 있습니다

---

### Step 6: 날짜 선택 입력 - Date Picker 사용
- **title**: 달력으로 특정 날짜 선택
- **text**: 날짜 네비게이션 중앙에는 날짜 입력 필드가 있습니다. 이 필드의 왼쪽에는 작은 📅 아이콘이 있고, 현재 선택된 날짜(예: "2025-04-07")가 텍스트로 표시됩니다. 이 필드를 클릭하면 네이티브 날짜 선택 달력이 열리므로, 원하는 날짜를 마우스로 클릭하여 바로 선택할 수 있습니다. 날짜를 선택하면 자동으로 그 날짜의 접수 현황이 로드되고, URL이 ?date=2025-04-07 형식으로 업데이트됩니다.
- **highlight**: `input[type="date"]` (날짜 입력 필드)
- **trigger**: click
- **tips**:
  - 달력 UI는 브라우저 기본 UI이므로, 환경에 따라 모습이 다를 수 있습니다
  - 모바일에서는 더 간단한 날짜 선택 UI가 표시됩니다
  - 매우 먼 과거 날짜를 선택해도 작동하지만, 데이터가 없으면 "접수된 내역이 없습니다"라는 메시지가 표시됩니다
  - 미래 날짜는 선택할 수 없습니다(예약과 달리)

---

### Step 7: 날짜 네비게이션 - 오늘로 이동 버튼
- **title**: 오늘 날짜로 빠르게 되돌아가기
- **text**: 날짜 입력 필드 옆에는 "오늘로 이동"이라는 파란 버튼이 있습니다. 이 버튼은 현재 선택된 날짜가 오늘이 아닐 때만 표시됩니다. 버튼을 클릭하면 즉시 오늘 날짜의 접수 현황으로 돌아갑니다. 이 기능은 과거 날짜를 여러 개 확인한 후, 빠르게 "오늘"로 돌아가고 싶을 때 유용합니다.
- **highlight**: `a[href="?date="]` (오늘로 이동 버튼)
- **trigger**: click
- **tips**:
  - 이 버튼이 표시되지 않으면, 현재 이미 오늘 날짜를 보고 있다는 뜻입니다
  - 퇴근 후 다음 날 아침에 업무를 시작할 때, 이 버튼을 클릭하면 최신 데이터로 리셋됩니다
  - 버튼 위에 마우스를 올리면 배경이 밝아져 클릭 가능함을 알립니다

---

### Step 8: 날짜 네비게이션 - 다음 날짜 버튼
- **title**: 다음 날짜로 이동
- **text**: 날짜 입력 필드 우측에 우측 화살표(→) 아이콘 버튼이 있습니다. 이 버튼을 클릭하면 하루 뒤 날짜의 접수 현황을 조회합니다. 예를 들어 현재 2025-04-07을 보고 있으면, 이 버튼을 클릭하면 2025-04-08의 데이터로 변경됩니다. 이 버튼을 반복 클릭하여 미래 날짜(최대 당일)까지 이동할 수 있습니다. 당일 다음 날(미래)은 데이터가 없으므로 "접수된 내역이 없습니다"라는 메시지가 표시됩니다.
- **highlight**: `a[href*="?date="]` (다음 화살표 링크)
- **trigger**: click
- **tips**:
  - 일반적으로 이 버튼은 사용할 일이 드뭅니다 (과거 날짜를 확인할 용도이므로)
  - 다음 날의 예약을 미리 확인하고 싶으면, 이 버튼으로 이동한 후 /admin/reservations 페이지를 확인하세요
  - 버튼을 클릭했을 때 데이터가 없으면, 그 날짜에 접수가 없다는 뜻입니다

---

### Step 9: 검색 폼 - 이름/연락처 검색
- **title**: 환자명 또는 연락처로 당일 접수자 검색
- **text**: 날짜 네비게이션 우측에는 검색 폼이 있습니다. 이 폼에는:
  - **입력 필드**: "이름/연락처 검색..." 플레이스홀더가 있는 텍스트 입력창
  - **검색 버튼**: "검색"이라고 쓰인 어두운 버튼 (bg-slate-900)
  
  환자의 이름(예: "김철수") 또는 연락처(예: "01012345678")를 입력한 후 엔터키를 누르거나 "검색" 버튼을 클릭하면, 현재 선택된 날짜 내에서 해당 환자만 필터링됩니다. 검색은 선택된 날짜 범위 내에서만 작동합니다 (다른 날짜의 환자는 검색되지 않음).
- **highlight**: `.relative.flex-1.md\\:w-64` (검색 입력 필드)
- **trigger**: input
- **tips**:
  - 검색어는 부분 일치로 작동합니다 (예: "김"만 입력하면 김씨 모두 검색)
  - 연락처는 하이픈 없이 숫자만 입력해도 됩니다
  - 검색 결과는 현재 선택된 날짜 내에서만 필터링됩니다
  - 검색 폼에는 숨겨진 `<input type="hidden" name="date" />`가 있어, 날짜 선택이 검색 후에도 유지됩니다

---

### Step 10: 접수 내역 테이블 - 환자 정보 확인
- **title**: 당일 접수된 환자 목록
- **text**: 페이지 중앙의 표에는 선택된 날짜의 모든 접수 내역이 표시됩니다. 테이블에는 다음 컬럼들이 있습니다:

  1. **접수 시간** (가장 왼쪽): "14:30" 형식의 접수 시간 (24시간 형식)
  
  2. **구분**: "초진" 또는 "재진" 배지
     - 초진: 초록 배지 (bg-emerald-100, text-emerald-800)
     - 재진: 파란 배지 (bg-blue-100, text-blue-800)
  
  3. **이름**: 첫 글자와 함께 환자 이름 (예: "김 / 김철수") + 그룹 아바타
  
  4. **정보**: 전화번호, 생년월일, 성별이 표시됨
     - 예: "010-1234-5678" (첫 줄)
     - "1980-05-15 • 남성" (두 번째 줄)
  
  5. **비고**: 접수 시 기입한 메모나 사유 (예: "등통증 악화", "정기 건강 관리")
  
  6. **관리** (가장 오른쪽): "차트 열기" 버튼 (파란 텍스트, 우측 화살표 아이콘)
  
  각 행은 호버 시 배경이 밝아져 클릭 가능함을 알립니다.
- **highlight**: `tbody` (테이블 바디)
- **trigger**: confirm
- **tips**:
  - 시간순으로 자동 정렬되어 있으므로, 오전에 들어온 환자부터 오후 순서로 나열됩니다
  - 초진과 재진의 색상을 구분하여, 신규 고객을 즉시 인식할 수 있습니다
  - 비고 항목을 통해 환자의 주증상을 한눈에 파악할 수 있으므로, 직원들이 진료 준비를 미리 할 수 있습니다

---

### Step 11: 차트 열기 버튼 - 환자 상세 페이지 이동
- **title**: 환자 차트 페이지로 빠르게 이동
- **text**: 각 행의 우측 끝에 "차트 열기" 버튼이 있습니다. 이 버튼을 클릭하면 그 환자의 상세 차트 페이지(/admin/patients/[id])로 즉시 이동합니다. 버튼은 파란 텍스트에 우측 화살표 아이콘이 있으며, 호버 시 배경이 파란색으로 변합니다. 이 기능을 통해 접수 현황에서 직접 환자의 전체 진료 기록, 결제 이력, 메모 등을 조회할 수 있습니다.
- **highlight**: `a[href="/admin/patients/"]` (차트 열기 링크)
- **trigger**: click
- **tips**:
  - 환자를 클릭한 행 어디를 클릭해도 "차트 열기" 버튼으로 이동합니다 (호버 상태에서 전체 행이 클릭 가능)
  - 상세 페이지에서 환자의 전체 진료 내용을 기록하고, 다음 방문 계획을 세울 수 있습니다
  - 접수 후 5분 이내에 차트를 열어 기본 진료 정보를 확인하는 습관을 들이세요

---

### Step 12: 접수 내역 없음 화면
- **title**: 검색 결과 없음 또는 접수 내역 없음
- **text**: 선택된 날짜에 접수 내역이 없으면, 테이블 자리에 큰 안내 화면이 표시됩니다:
  - 중앙에 큰 📭 아이콘 (우편함)
  - "접수된 내역이 없습니다"라는 제목
  - "해당 날짜에 접수된 초진/재진 기록이 없습니다."라는 설명

  검색어를 입력했는데 결과가 없으면, 같은 화면이 표시됩니다 (이는 그 날짜에 검색 조건에 맞는 환자가 없다는 뜻).
- **highlight**: `td[colspan="6"]` (빈 상태 메시지)
- **trigger**: confirm
- **tips**:
  - 주말이나 휴무일에는 당연히 이 메시지가 표시됩니다
  - 휴무일인 줄 모르고 날짜를 선택한 경우, 이전/다음 화살표로 인근 영업일을 확인하세요
  - 검색 조건을 너무 구체적으로 입력하면 (예: 정확한 전화번호), 결과가 없을 수 있으므로, 부분 검색을 시도하세요

---

### Step 13: 모바일 최적화 안내
- **title**: 모바일에서의 접수 관리
