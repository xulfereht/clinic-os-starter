## 3. 왜 Clinic-OS인가? (기존 section-tag: features)

**section-tag**: `why_clinic_os`

**헤드라인**: 왜 Clinic-OS인가?

**3컬럼 카드:**

**카드 1 — AI 파트너**
```
에이전트가 함께합니다

블로그 글 3시간 → 15분.
월간 보고서? 말 한마디.
환자 리마인드? 자동.
원장님의 시간을 진료에 돌려드립니다.
```

**카드 2 — 쓸수록 성장**
```
데이터가 쌓이면 더 똑똑해집니다

환자 패턴 분석, 이탈 예측, 시간대별 최적화 —
쓰면 쓸수록 에이전트의 조언이 정확해집니다.
새 기능도 자동으로 업데이트됩니다.
```

**카드 3 — 완전한 소유**
```
내 데이터, 내 서버, 내 시스템

월 구독료 없음. 플랫폼 종속 없음.
내 Cloudflare 계정에서 직접 운영.
원하면 소스코드 수정도 자유롭게.
```

> **NOTE**: "소유" 메시지는 유지하되, 3번째(부가 가치)로 내린다. 1순위는 에이전트 파트너십.

## 4. 에이전트가 하는 일 (기존 section-tag: agent_workflow)

**section-tag**: `what_agent_does`

**헤드라인**:
```
말하면 됩니다. 에이전트가 합니다.
```

**서브헤드**:
```
코드를 배울 필요 없습니다. 상황과 원하는 결과를 말하면,
에이전트가 진단하고, 계획하고, 실행합니다.
```

**3-4개 대화 예시 (터미널 스타일):**

**예시 1 — 경영 분석**
```
"이번 달 매출 분석해줘"
→ 환자별, 프로그램별, 시간대별 매출 분석
→ 전월 대비 변화 + 추천 액션
```

**예시 2 — 콘텐츠 생산**
```
"허리 통증 관련 블로그 글 써줘"
→ 원장님 톤으로 1,500자 초안 작성
→ 의료광고 심의 기준 자동 검토
→ 발행까지 원클릭
```

**예시 3 — 환자 관리**
```
"60일 이상 안 온 환자들에게 리마인드 보내줘"
→ 대상 자동 추출 (32명)
→ 메시지 초안 생성
→ 원장님 확인 후 발송
```

**예시 4 — 홈페이지 수정**
```
"홈페이지 메인 문구 바꿔줘"
→ 기존 문구 확인
→ 변경 적용 + 빌드 + 배포
→ 1분 만에 반영 완료
```

## 5. 스킬 생태계 (기존 plugins 섹션 대체/확장)

**section-tag**: `skill_ecosystem`

**헤드라인**:
```
53개 스킬. 계속 늘어납니다.
```

**서브헤드**:
```
에이전트의 능력은 '스킬'로 확장됩니다.
새 스킬이 개발되면 자동으로 배포되어, 내 에이전트가 더 똑똑해집니다.
```

**스킬 카테고리 그리드:**

| 카테고리 | 예시 스킬 | 설명 |
|----------|-----------|------|
| 경영 분석 | 경영 자문, 월간 보고서 | 데이터 기반 인사이트 |
| 환자 관리 | 세그먼트 분석, 리마인드 | 이탈 방지, 재방문 유도 |
| 콘텐츠 | 블로그 작성, FAQ 생성 | 원장님 톤으로 자동 생산 |
| 마케팅 | 카피 작성, 캠페인, 후기 관리 | 광고 심의까지 자동 검토 |
| 사이트 빌드 | 홈페이지, 프로그램, 스킨 | 대화로 디자인 변경 |
| 확장 | 플러그인, 검사도구 | 원하는 기능 직접 추가 |

**하단 Tip Box:**
```
스킬은 정기적으로 업데이트됩니다.
새로 개발된 스킬은 자동으로 배포되어 — 별도 설치 없이 바로 사용 가능합니다.
```

```

"완성된 시스템을 받아, 운영에 집중합니다"

```

✓ 4주 온보딩 과정
✓ 코어 업데이트 + 스킬 자동 배포
✓ 커뮤니티 + 가이드 접근

**section-tag**: `onboarding_journey`

**헤드라인**:
```
4주 온보딩 여정
```

**서브헤드**:
```
코딩 교육이 아닙니다.
에이전트와 협업하는 방법을 익히는 과정입니다.
```

**4주 타임라인:**

```
1주차: 에이전트 만나기
  → 시스템 설치 + 기본 정보 입력 + 첫 배포
  → "내 사이트가 바로 열립니다"

2주차: 콘텐츠 채우기
  → 네이버 블로그/플레이스 자동 이전 + 프로그램 등록
  → "기존 콘텐츠가 새 사이트로 옮겨집니다"

3주차: 강점 발굴
  → 차별점 분석 + 카피 작성 + 홈페이지 완성
  → "우리 한의원만의 메시지가 완성됩니다"

4주차: 운영 시작
  → 환자 분석 + 캠페인 + 경영 리포트
  → "에이전트가 경영 파트너로 작동하기 시작합니다"
```

## 8. FAQ (기존 section-tag: faq)

**Q: 코딩을 몰라도 되나요?**
A: 네. 코드를 볼 일이 없습니다. 에이전트에게 대화로 지시하면 됩니다.

**Q: 기존 홈페이지가 있는데요?**
A: 네이버 블로그, 플레이스 정보를 자동으로 가져옵니다. 기존 콘텐츠를 잃지 않습니다.

**Q: 월 유지비가 있나요?**
A: Cloudflare 무료 플랜으로 운영됩니다. 별도 월 구독료 없습니다.
   AI 에이전트(Claude Pro) 구독만 별도입니다 (월 $20, 선택).

**Q: 소스코드를 받는 건가요?**
A: 네. 전체 소스코드를 소유합니다. 하지만 중요한 건 소스코드가 아니라 —
   그 위에서 돌아가는 에이전트 시스템입니다. 코드는 수단이고, 에이전트가 가치입니다.

**Q: 업데이트는 어떻게 되나요?**
A: 새 기능(스킬)과 시스템 개선이 자동으로 배포됩니다.
   내 커스터마이징은 그대로 보존됩니다.

**Q: 위임 셋업을 하면 이후 수정은?**
A: 인수 후에는 에이전트와 대화로 직접 수정합니다.
   홈페이지 문구 변경, 프로그램 추가, 블로그 작성 — 전부 대화로 됩니다.
