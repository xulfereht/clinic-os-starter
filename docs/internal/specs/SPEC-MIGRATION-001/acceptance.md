## 7. 롤백 트리거

다음 상황 발생 시 즉시 롤백:

1. **데이터 손실/불일치** 감지
2. **핵심 기능 장애** (예약, 결제, 로그인)
3. **5분 이상 서비스 불가**
4. **보안 취약점** 발견

롤백 절차:
```bash
# 도메인 전환 방식 사용 시
# Cloudflare Dashboard → DNS → brd-clinic.com CNAME → 기존 프로젝트로 변경

# 또는 기존 레포에서 재배포
cd ~/dev/brd-clinic
npm run build
npx wrangler pages deploy dist --project-name=brd-clinic
```
