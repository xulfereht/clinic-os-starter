# Clinic-OS 사용자 가이드

> 한의원 웹사이트 운영을 위한 상세 가이드

---

## 📱 퍼블릭 페이지 (환자용)

### 메인 페이지 (`/`)
- **용도**: 한의원 첫인상, 핵심 메시지 전달
- **구성**: 히어로 섹션, 진료 소개, 철학, 정보
- **수정**: 코드 수정 필요 (향후 관리자에서 편집 가능 예정)

### 진료 프로그램 (`/programs`)
- **용도**: 제공하는 진료 프로그램 목록
- **수정**: `/admin/programs`에서 편집

### 프로그램 상세 (`/programs/[id]`)
- **용도**: 개별 프로그램 상세 정보
- **구성**: 히어로, 문제/솔루션, 치료과정, 가격, FAQ 등
- **수정**: `/admin/programs/[id]`에서 섹션별 편집

### 의료진 소개 (`/doctors`)
- **용도**: 원장님/의료진 프로필
- **수정**: `/admin/doctors`에서 편집

### 오시는 길 (`/location`)
- **용도**: 한의원 위치, 교통 안내
- **수정**: `/admin/settings`에서 주소/지도 설정

### 블로그/칼럼 (`/blog`)
- **용도**: 건강 정보, 치료 사례 등 콘텐츠
- **수정**: `/admin/posts`에서 작성/편집

### 치료 후기 (`/reviews`)
- **용도**: 환자 후기 모음
- **수정**: `/admin/reviews`에서 관리

### 비대면 진료 (`/telemedicine`)
- **용도**: 원격 진료 안내 (선택 기능)
- **활성화**: `/admin/settings` → 비대면 진료 ON

### 문진표 (`/intake`)
- **용도**: 환자 사전 문진
- **수정**: `/admin/intake`에서 항목 편집

## 🔐 로그인 방법

### 관리자 로그인
1. `/admin/login` 접속
2. 이메일/비밀번호 입력
3. 로그인 버튼 클릭

### 기본 계정 (초기 설정 시)
```
이메일: admin@sample-clinic.com
비밀번호: admin123
```

> ⚠️ **중요**: 첫 로그인 후 반드시 비밀번호를 변경하세요!

### 비밀번호 변경
1. `/admin/settings` 접속
2. 계정 설정에서 비밀번호 변경

### 새 관리자 추가
1. `/admin/staff` 접속
2. "직원 추가" 클릭
3. 권한 설정 (관리자/스태프)

## ⚙️ 관리자 기능 상세

### 📊 대시보드 (`/admin`)

**표시 정보:**
- 오늘의 예약 현황
- 최근 환자 등록
- 주요 지표 (방문자, 문의 등)

**Antigravity에게 요청:**
- "대시보드 통계 기간 변경해줘"
- "새로운 지표 추가해줘"

---

### 👥 환자 관리 (`/admin/patients`)

**기능:**
- 환자 목록 조회/검색
- 환자 상세 정보 확인
- 진료 기록 관리
- 결제 내역

**사용법:**
1. 환자 이름/전화번호로 검색
2. 환자 클릭 → 상세 페이지
3. 탭으로 정보 전환 (기본정보/진료/결제)

**Antigravity에게 요청:**
- "환자 검색 필터 추가해줘"
- "엑셀 내보내기 기능 만들어줘"

---

### 📅 예약 관리 (`/admin/reservations`)

**기능:**
- 예약 목록 (일별/주별)
- 예약 상태 관리 (확정/취소/완료)
- 새 예약 등록

**사용법:**
1. 날짜 선택
2. 예약 클릭 → 상세/수정
3. 상태 변경 버튼 클릭

**Antigravity에게 요청:**
- "예약 알림 설정 바꿔줘"
- "캘린더 뷰 추가해줘"

---

### 🏥 진료 프로그램 (`/admin/programs`)

**기능:**
- 프로그램 목록 관리
- 프로그램 상세 편집 (섹션별)
- 가격/FAQ 설정

**프로그램 편집 방법:**
1. 프로그램 선택
2. 왼쪽: 섹션 목록
3. 오른쪽: 선택한 섹션 편집
4. 실시간 미리보기 확인
5. "저장하기" 클릭

**섹션 타입:**
| 타입 | 설명 |
|------|------|
| Hero | 상단 이미지/제목 |
| Problem | 문제점 설명 |
| Solution | 해결책 제시 |
| Process | 치료 과정 |
| Pricing | 가격표 |
| FAQ | 자주 묻는 질문 |
| DoctorIntro | 담당 의료진 |

---

### 📝 블로그/칼럼 (`/admin/posts`)

**기능:**
- 글 작성/편집
- 카테고리 분류
- 발행/비발행 설정

**글 작성 방법:**
1. "새 글 작성" 클릭
2. 제목, 내용 입력
3. 썸네일 이미지 업로드
4. 카테고리 선택
5. "발행하기" 클릭

---

### ⭐ 후기 관리 (`/admin/reviews`)

**기능:**
- 환자 후기 목록
- 후기 승인/비공개 설정
- 답글 작성

**사용법:**
1. 후기 목록 확인
2. 공개할 후기 선택
3. "공개" 상태로 변경
4. 필요시 답글 작성

---

### 💬 메시지/알림톡 (`/admin/messages`)

**기능:**
- 환자 메시지 관리
- AlimTalk 템플릿 관리
- 발송 내역 확인

**AlimTalk 사용법:**
1. `/admin/settings/alimtalk`에서 API 키 설정
2. 템플릿 등록/승인
3. 환자 선택 → 메시지 발송

---

### ⚙️ 설정 (`/admin/settings`)

**주요 설정 항목:**

| 메뉴 | 설명 |
|------|------|
| 기본 정보 | 한의원 이름, 연락처, 주소 |
| 운영 시간 | 영업시간, 휴무일 |
| SEO | 검색엔진 최적화 설정 |
| 다국어 | 다국어 지원 ON/OFF |
| 비대면 | 원격 진료 기능 ON/OFF |
| 연동 | 외부 서비스 (카카오, AlimTalk 등) |

## 🔧 자주 하는 작업

### 한의원 이름 변경
1. `/admin/settings` 접속
2. "기본 정보" 탭
3. 한의원 이름 수정
4. 저장

### 로고 변경
1. `/admin/settings` 접속  
2. "디자인" 탭
3. 로고 이미지 업로드
4. 저장

### 새 프로그램 추가
1. `/admin/programs` 접속
2. "새 프로그램" 버튼
3. 기본 정보 입력
4. 섹션별 내용 추가
5. 저장 후 발행

### 공지사항 등록
1. `/admin/notices` 접속
2. "새 공지" 버튼
3. 제목/내용 입력
4. 발행

## ❓ FAQ

### Q: 수정한 내용이 바로 반영 안 돼요
→ 브라우저 캐시 때문일 수 있습니다. `Ctrl+Shift+R`로 새로고침하세요.

### Q: 이미지가 안 올라가요
→ 파일 크기가 2MB 이하인지 확인하세요. JPG/PNG 형식만 지원됩니다.

### Q: 로그인이 안 돼요
→ 초기 계정 `admin@sample-clinic.com / admin123`으로 시도하세요.

### Q: 메뉴가 안 보여요
→ 권한 문제일 수 있습니다. 관리자 계정인지 확인하세요.
