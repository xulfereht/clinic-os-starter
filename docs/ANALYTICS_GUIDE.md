# 분석 및 통계 가이드

이 가이드는 대시보드와 통계 페이지를 통해 병원의 운영 현황을 파악하고 의사결정에 활용하는 방법을 설명합니다.

## 1. 주요 지표 (KPI)
- **방문자 수**: 홈페이지를 방문한 사용자 수 및 유입 경로를 확인합니다.
- **예약 전환율**: 검색 또는 방문이 실제 예약으로 이어진 비율을 분석합니다.
- **신환/구환 비율**: 신규 환자와 재방문 환자의 비율을 통해 병원의 성장세를 파악합니다.

## 2. 예약 통계
- **요일/시간대별 분석**: 어느 시간대에 환자가 집중되는지 파악하여 인력 배치나 진료 시간을 조정할 수 있습니다.
- **프로그램별 인기순위**: 어떤 진료 프로그램이 가장 인기가 많은지 확인하여 마케팅 집중도를 결정합니다.

## 3. 유입 경로 분석 (AEO/SEO)
- **검색 키워드**: 환자들이 어떤 키워드로 우리 병원을 찾는지 확인합니다.
- **광고 성과**: 발송된 알림톡이나 문자 메시지를 통해 실제 예약이 얼마나 발생했는지 기여도를 측정합니다.

## 4. 리포트 내보내기
- 필요한 통계 데이터를 엑셀(CSV) 형식으로 다운로드하여 내부 보고서 작성에 활용할 수 있습니다.

> [!TIP]
> 통계 데이터는 매일 자정 기준으로 최신화됩니다. 실시간 예약 상황은 '예약 관리' 페이지에서 확인하세요.
