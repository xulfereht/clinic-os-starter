# 메시지 및 캠페인 관리 가이드

이 가이드는 알리고(Aligo) 서비스를 연동하여 환자에게 예약 안내, 알림톡, 마케팅 문자를 발송하고 템플릿을 관리하는 방법을 설명합니다.

---

## 1. 메시지 서비스 연동 확인

메시지 발송을 위해서는 먼저 외부 서비스인 **알리고(Aligo)** 연동이 완료되어야 합니다.
- **연동 확인**: [시스템 설정] > [API 연동] 메뉴에서 API 키와 발신번호가 등록되어 있는지 확인하세요.
- **잔액 확인**: 알리고 홈페이지에서 충전된 잔액이 있어야 문자가 정상적으로 발송됩니다.

---

## 2. 메시지 템플릿 관리

반복적으로 사용하는 문구는 템플릿으로 등록하여 효율적으로 사용할 수 있습니다.

### 📱 문자 (SMS/LMS) 템플릿
1.  **[메시지 템플릿]** 메뉴에서 **[새 템플릿]**을 클릭합니다.
2.  이름과 카테고리(예약, 안내 등)를 정하고 내용을 작성합니다.
3.  **변수 활용**: `{name}`, `{reservation_datetime}` 등의 변수를 본문에 넣으면, 발송 시 실제 환자 이름과 예약 시간으로 자동 치환됩니다.

### 💬 알림톡 (카카오톡) 템플릿
알림톡은 카카오의 승인을 받은 문구만 발송 가능합니다.
1.  알리고 관리자 페이지에서 템플릿을 등록하고 승인을 받습니다.
2.  시스템의 **[메시지 템플릿]** 페이지에서 **[알림톡 동기화]** 버튼을 누릅니다.
3.  승인된 템플릿이 자동으로 불러와지며, 이후 캠페인에서 선택하여 사용 가능합니다.

---

## 3. 문자 캠페인 발송 방법 (단체 발송)

특정 그룹에게 정보를 전달하거나 마케팅 문자를 보낼 때 사용합니다.

1.  **[문자 캠페인]** 메뉴에서 **[새 캠페인 작성]**을 클릭합니다.
2.  **대상 선택**: 전체 환자 또는 특정 세그먼트(예: 3개월 내 방문 고객)를 선택합니다.
3.  **템플릿 선택**: 미리 만들어둔 템플릿을 불러옵니다.
4.  **발송 시점**: '즉시 발송' 또는 '예약 발송'을 선택합니다.
5.  **필터링**: '수신 거부 고객 제외', '중복 번호 제외' 등의 옵션을 확인합니다.
6.  **발송**: 최종 확인 후 발송 버튼을 누릅니다.

---

## 4. 자동 발송 설정 (시스템 메시지)

예약 확정이나 전날 안내와 같이 시스템에서 자동으로 나가는 메시지 설정입니다.
- **예약 안내**: 환자가 홈페이지에서 예약 시 즉시 발송됩니다.
- **리마인드**: 예약 1일 전 또는 당일 아침에 자동으로 발송되도록 설정할 수 있습니다.
- 설정 변경은 **[기본 설정]** 또는 **[예약 관리]** 내의 메시지 설정 옵션을 확인하세요.

---

## 💡 유용하게 사용되는 메시지 변수 목록
- `{name}`: 고객/환자 성함
- `{clinic_name}`: 우리 병원 이름
- `{clinic_phone}`: 병원 대표 전화번호
- `{reservation_datetime}`: 예약 날짜 및 시간
- `{bank_info}`: [기본 설정]에 등록된 입금 계좌 정보

---

## ⚠️ 주의사항
- **야간 발송 제한**: 정보통신망법에 따라 광고성 정보는 야간(오후 9시 ~ 다음 날 오전 8시) 발송이 금지되어 있습니다.
- **수신 거부 처리**: 마케팅 캠페인 발송 시 반드시 수신 거부(080) 번호가 포함되어야 하며, 거부 의사를 밝힌 고객은 리스트에서 제외해야 합니다.
