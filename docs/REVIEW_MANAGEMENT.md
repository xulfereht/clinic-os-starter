# 리뷰 관리 가이드

이 가이드는 홈페이지 및 외부 서비스(영수증 리뷰 등)를 통해 수집된 환자들의 소중한 리뷰를 관리하고 활용하는 방법을 설명합니다.

## 1. 리뷰 목록 확인
- **최신 리뷰**: 등록된 순서대로 모든 리뷰를 한눈에 볼 수 있습니다.
- **필터링**: 미답변 리뷰, 특정 별점 이하 리뷰, 포토 리뷰만 모아보기 등을 통해 빠른 대응이 가능합니다.

## 2. 리뷰 답변 및 관리
- **공식 답변 등록**: 환자의 리뷰에 감사 인사 또는 안내 메시지를 등록할 수 있습니다. 
- **공개 설정**: 부적절한 광고성 리뷰나 비방글은 관리자 권한으로 숨김 처리하거나 검토 요청을 할 수 있습니다.

## 3. 대표 리뷰 설정
- **기능**: 환자들에게 가장 먼저 보여주고 싶은 우수 리뷰를 '대표 리뷰'로 지정하여 홈페이지 메인 또는 프로그램 상세 페이지 상단에 고정할 수 있습니다.
- **방법**: 리뷰 우측의 ⭐ 아이콘을 클릭하여 활성화합니다.

## 4. 리뷰 요청 알림톡
- **자동 발송**: 진료 완료 후 환자에게 리뷰 작성을 요청하는 알림톡을 자동으로 발송하도록 설정할 수 있습니다. (메시지 캠페인 관리와 연동)

> [!IMPORTANT]
> 진정성 있는 답변은 잠재 고객의 신뢰를 얻는 가장 효과적인 방법입니다. 불만 리뷰에도 정중하고 신속하게 대응하는 것을 권장합니다.
