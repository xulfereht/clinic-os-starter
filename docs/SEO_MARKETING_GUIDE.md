# SEO 및 마케팅 설정 가이드

이 가이드는 검색 엔진 최적화(SEO)와 외부 마케팅 도구를 연동하여 더 많은 환자에게 병원을 노출하는 방법을 설명합니다.

## 1. 기본 SEO 설정
- **사이트 제목 및 설명**: 네이버, 구글 검색 결과에 표시될 병원 이름과 설명을 설정합니다.
- **대표 키워드**: 병원을 상징하는 핵심 키워드(예: 강남한의원, 추나요법, 다이어트한약 등)를 등록합니다.

## 2. 네이버 서치어드바이저 / 구글 서치콘솔
- **사이트 소유권 확인**: 각 포털의 소유권 확인 메타 태그를 입력하여 사이트 수집을 요청합니다.
- **사이트맵(Sitemap) 자동 생성**: Clinic-OS는 검색 봇이 사이트를 잘 파악할 수 있도록 사이트맵을 항상 최신으로 유지합니다.

## 3. 소셜 미디어 미리보기 (Open Graph)
- 카카오톡이나 페이스북 등에 홈페이지 링크를 공유했을 때 나타나는 이미지와 문구를 설정합니다. 이벤트 기간에는 이벤트 배너로 변경하여 주목도를 높이세요.

## 4. 분석 도구 연동 (Pixel / GA4)
- **메타 픽셀**: 페이스북/인스타그램 광고 성과를 측정하기 위한 코드를 삽입합니다.
- **구글 애널리틱스 (GA4)**: 상세한 사용자 행동 분석을 위한 측정 ID를 연동합니다.

> [!TIP]
> SEO 설정은 변경 후 실제 검색 엔진에 반영되기까지 며칠에서 몇 주가 소요될 수 있습니다. 인내심을 갖고 꾸준히 콘텐츠를 업로드하는 것이 가장 중요합니다.
