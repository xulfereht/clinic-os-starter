# 접수 현황 관리 가이드

이 가이드는 일별 환자 내원 기록을 확인하고, 통계 데이터를 통해 진료 현황을 분석하는 방법을 설명합니다.

## 1. 일별 접수 목록
- **실시간 내원 확인**: '예약 관리'에서 내원 확인(진료 시작) 처리가 된 환자들이 시간순으로 정렬되어 표시됩니다.
- **초진/재진 분류**: 신규 환자와 재방문 환자가 자동으로 구분되어 집계됩니다.

## 2. 현황 통계 (Stats)
- 금일 총 내원객 수, 초진 환자 수, 재진 환자 수를 상단 위젯을 통해 직관적으로 파악할 수 있습니다.

## 3. 상세 내역 조회
- 목록에서 환자 이름을 클릭하면 해당 환자의 상세 차트로 이동하여 과거 기록을 함께 검토할 수 있습니다.

## 4. 데이터 활용
- 접수 데이터를 기반으로 특정 요일이나 시간대의 혼잡도를 파악하여 병원 운영 효율화에 활용하세요.

> [!NOTE]
> 접수 현황은 실제 '진료실 입실' 시점을 기준으로 기록되는 것이 가장 정확합니다. 데스크 업무 프로세스에 맞춰 사용하세요.
