# 예약 관리 가이드

이 가이드는 일별 예약 현황을 한눈에 파악하고, 새로운 예약을 등록하거나 기존 일정을 관리하는 방법을 설명합니다.

## 1. 예약 현황판 (캘린더/리스트)
- **날짜별 조회**: 특정 날짜를 선택하여 실시간 예약 상황을 확인합니다.
- **필터링**: 담당의별, 예약 상태별(대기, 완료, 취소 등)로 모아볼 수 있습니다.

## 2. 예약 등록 및 수정
- **환자 연결**: 이름이나 전화번호 검색을 통해 기존 환자 정보를 불러온 뒤 예약을 등록합니다.
- **시간 슬롯**: 병원의 설정된 진료 시간 내에서 예약 가능한 잔여 슬롯을 선택합니다.

## 3. 상태 업데이트 및 내원 확인
- 환자가 도착하면 '내원 확인' 버튼을 클릭하여 상태를 변경합니다. 이는 나중에 '접수 현황' 데이터로 자동 집계됩니다.

## 4. 알림 메시지 발송
- 예약이 확정되거나 변경되었을 때 환자에게 카카오톡 알림을 즉시 발송할 수 있습니다.

> [!TIP]
> '드래그 앤 드랍' 형태로 예약 시간을 직관적으로 변경할 수 있는 기능을 활용하면 전화 응대 시 더 빠르게 일정을 조정할 수 있습니다.
