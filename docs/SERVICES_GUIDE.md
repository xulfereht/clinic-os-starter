# 🌐 외부 서비스 연동 가이드

Clinic-OS의 핵심 기능(서버 배포, 데이터베이스, 알림톡 등)을 활성화하기 위해 필요한 외부 서비스 계정 및 설정 목록입니다.

---

## ☁️ 1. Cloudflare (필수 - 인프라)

Clinic-OS의 서버 배포, 데이터베이스(D1), 이미지 서버(R2)를 담당합니다.

- **가입 주소**: [cloudflare.com](https://www.cloudflare.com/)
- **필요 요금제**: 무료(Free) 티어로 충분히 시작 가능
- **준비해야 할 정보**:
  - `Account ID`: 대시보드 우측 하단에서 확인 가능
  - `API Token`: [사용자 프로필 > API 토큰]에서 `Edit Cloudflare Workers` 템플릿으로 생성

---

## 💬 2. 알리고 (선택 - 알림톡/SMS)

환자에게 예약 안내, 상담 완료 메시지를 보낼 때 사용합니다.

- **가입 주소**: [smartsms.aligo.in](https://smartsms.aligo.in/)
- **중요**: 
  - 가입 후 **본인 인증** 및 **발신 번호 등록**이 필요합니다.
  - 알림톡을 사용하려면 **카카오 채널 연동** 및 **템플릿 심사**를 거쳐야 합니다.
- **준비해야 할 정보**:
  - `API Key`: 알리고 설정 메뉴에서 발급
  - `User ID`: 알리고 로그인 아이디
  - `Sender Number`: 승인된 발신 번호

---

## 🏷️ 3. 가비아/후이즈 (선택 - 도메인)

한의원 고유의 도메인(예: `abc-clinic.com`)을 사용하고 싶을 때 필요합니다.

- 가비아, 후이즈 등에서 도메인을 구매하신 후 Cloudflare 네임서버로 연결해야 합니다.

---

## 🏁 서비스 연동 체크리스트

설정 마법사(`/setup-clinic`)를 실행하기 전 아래 항목들이 모두 준비되었는지 확인하세요.

- [ ] Cloudflare 계정 가입 완료
- [ ] Cloudflare API 토큰 발급 완료 (Worker 편집 권한)
- [ ] 알리고 가입 및 발신 번호 승인 완료 (문자 사용 시)
- [ ] 카카오 알림톡 템플릿 심사 완료 (알림톡 사용 시)
