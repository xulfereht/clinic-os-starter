# 메시지 템플릿 가이드

이 가이드는 알림톡 및 문자 발송에 사용될 템플릿을 등록하고 관리하는 방법을 설명합니다.

## 1. 알리톡 템플릿 등록
- **카카오 심사**: 알림톡 템플릿은 카카오의 사전 심사가 필요합니다. 템플릿 내용과 버튼 구성을 입력한 후 '심사 요청'을 진행하세요.
- **변수 활용**: `#{이름}`, `#{예약일시}` 와 같은 변수를 사용하여 환자별 맞춤 메시지를 구성할 수 있습니다.

## 2. 문자(SMS/LMS) 템플릿 관리
- 별도의 심사 없이 자유롭게 작성하여 사용할 수 있습니다.
- 이미지 포함 여부에 따라 문자 유형이 자동으로 전환됩니다.

## 3. 템플릿 동기화
- '알리거(Aligo) 동기화' 버튼을 클릭하면 외부 플랫폼에 등록된 최신 템플릿 상태를 Clinic-OS로 가져옵니다.

> [!IMPORTANT]
> 알림톡 심사에는 영업일 기준 1~3일이 소요될 수 있습니다. 중요한 캠페인이 있다면 미리 템플릿을 등록해 주세요.
