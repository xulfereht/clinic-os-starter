# AI 기능 활용 가이드

Clinic-OS에서 제공하는 인공지능(AI) 도구를 사용하여 콘텐츠 제작 시간을 단축하고 전문성을 높이는 방법을 설명합니다.

## 1. AI 질문 생성 (자가진단용)
- **기능**: 진료 프로그램의 핵심 키워드만 입력하면, 환자가 답변하기 쉬운 문항과 선택지를 AI가 자동으로 생성합니다.
- **활용**: 새로운 다이어트 프로그램이나 통증 치료 프로그램을 설계할 때 기본 문항 베이스로 활용하세요.

## 2. 블로그/칼럼 글쓰기 어시스턴트
- **개요 작성**: 제목만 입력하면 AI가 논리적인 글의 목차(개요)를 잡아줍니다.
- **문체 변경**: 작성한 초안을 '친절한 의사 선생님 버전', '전문적인 정보 전달 버전' 등 원하는 톤으로 다듬어줍니다.

## 3. SEO 최적화 제안
- 작성한 글이 네이버나 구글 검색에 잘 노출될 수 있도록 핵심 키워드를 추천하고 메타 설명을 자동으로 생성해줍니다.

## 4. AI 설정값 조정
- **창의성 수준**: AI가 얼마나 창의적으로(또는 사실적으로) 답변할지 설정할 수 있습니다.
- **전문성 수준**: 일반인 대상 또는 전문가 대상 등 독자층에 따른 답변 수준을 선택합니다.

> [!CAUTION]
> AI가 생성한 내용은 항상 전문가(의료진)의 최종 검토를 거쳐야 합니다. 의학적 정확성은 반드시 사람이 확인하세요.
